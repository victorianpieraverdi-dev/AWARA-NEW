## ✅ Что проверить

### 1. **Кнопка Земли (🌍)**
- **Где:** left-bottom, зелёная
- **Должна быть видна сразу** (без условий)
- **Клик:** переход на `earth-player.html`
- **Если не видна:** открой DevTools → Console → ищи ошибки

### 2. **Кнопка Искры (✦)**
- **Где:** top-right, золотая
- **Должна появиться после:** завершения онбординга или при старте (если онбординг пройден)
- **Текст меняется по времени:**
  - 17:00–05:00 → "ВЕЧЕРНЯЯ ИСКРА"
  - 05:00–12:00 → "УТРЕННЯЯ ИСКРА"
  - 12:00–17:00 → "✦ ИСКРА"
- **Проверка в Console:**
  ```
  🔥 showSparkBtn called, hour: 20
  ✅ Spark button shown, mode: evening
  ```
- **Если не видна:** проверь `localStorage.awara_tutorial_completed_v2` — должно быть `'1'`

### 3. **Кнопка Тигель (✦ ПЕРЕЙТИ В ТИГЕЛЬ)**
- **Где:** right-bottom, золотая
- **Появляется после:** заполнения 3+ сфер (кроме Души)
- **Клик:** 
  1. Собирает записи за сегодня
  2. Сохраняет в `localStorage.awara_soul_day_context`
  3. Переход на `tigel.html`
- **Проверка в Console:**
  ```
  🔥 goToTigel called
  📊 Day context: {date: "2026-05-28", spheres: [...]}
  ✅ День передан в Тигель: {...}
  ```

### 4. **Мобильная панель**
- **Как проверить:** DevTools → Toggle device toolbar (Ctrl+Shift+M) → выбери iPhone/Android
- **Что должно быть:** панель сферы выходит **сверху** (не справа)
- **Ширина экрана:** ≤768px

### 5. **Вечерняя Искра (модальное окно)**
- **Клик на кнопку Искры** → открывается модальное окно
- **Содержимое:**
  - Заголовок: "ВЕЧЕРНИЙ ОГОНЬ · ДЕНЬ" (или утренний/полный)
  - ИНСАЙТ — текст резюме
  - СОВЕТЫ — 3 пункта с эмодзи
  - ВОПРОС — текст для медитации
- **Закрытие:** кнопка × (top-right в модалке)

## 🐛 Если что-то не работает

### Кнопки не видны
1. Открой DevTools → Console
2. Проверь ошибки JS
3. Проверь `localStorage`:
   ```js
   localStorage.getItem('awara_tutorial_completed_v2')
   localStorage.getItem('awara_subspheres_v3')
   ```

### Онбординг не пропускается
- Если хочешь пропустить онбординг:
  ```js
  localStorage.setItem('awara_tutorial_completed_v2', '1')
  location.reload()
  ```

### Кнопка Тигель не появляется
- Нужно заполнить 3+ сферы (кроме Души)
- Проверь в Console:
  ```js
  // Посмотри, сколько сфер заполнено
  const subSpheres = JSON.parse(localStorage.getItem('awara_subspheres_v3') || '{}');
  console.log(subSpheres);
  ```

### Синтез не виден в Тигеле
- **Проблема:** `tigel.html` НЕ читает `awara_soul_day_context`
- **Нужно:** добавить код в `tigel.html`, который читает этот ключ и показывает
- **Временное решение:** открой DevTools в `tigel.html` и проверь:
  ```js
  const ctx = localStorage.getItem('awara_soul_day_context');
  console.log('День из Души:', JSON.parse(ctx));
  ```
