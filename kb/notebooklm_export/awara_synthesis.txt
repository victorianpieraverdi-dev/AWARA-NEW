# Игровой поток (Game Flow)

Создано: 2026-05-24

## Путь игрока

```
ВХОД (Alpha Gate)
  ↓
ЛОББИ (Золотое Яйцо)
  ├─ ИГРА
  │   ├─ Матрицы (выбор восприятия)
  │   ├─ Карты (глобус, ведический космос, яйцо)
  │   ├─ Уровни
  │   │   ├─ Коридор Инициации (13 шагов)
  │   │   ├─ Пространство Души (3 сферы)
  │   │   ├─ Земля Игрока (Васту)
  │   │   ├─ Вселенная (агенты на орбитах)
  │   │   └─ Супер Игра (Мироздание)
  │   └─ Тигель (лог дня)
  ├─ ХРОНИКА
  │   ├─ Смысл дня
  │   ├─ Паспорт Души
  │   ├─ Колода Карт
  │   └─ Натальная Карта
  ├─ ОБМЕН
  │   ├─ Светкоин
  │   ├─ Милость
  │   ├─ Даймон
  │   └─ Артефакты
  └─ ОБЩЕНИЕ
      ├─ Оракул
      ├─ Чат Матрицы
      └─ Фестивали
```

## Ежедневный цикл
1. Зайти → Ключ Дня (агент + матрица + стихия)
2. Тигель → записать день → свет
3. Оракул → совет
4. Паспорт → прогресс

## Слои реальности
1. Аура человека (чакры, меры)
2. Регион / локация
3. Додекаэдр Земли (62 узла)
4. Чатуралока
5. Брахманда (14 лок)
6. Космические силы
7. Эпилог / возврат

# Связь Лок и Чакр

Создано: 2026-05-24

## Как читать
Каждая из 14 лок соответствует определённому уровню сознания. 
7 верхних лок соотносятся с 7 классическими чакрами (Муладхара → Сахасрара).
7 нижних лок — с теневыми аспектами тех же чакр.

## Верхние локи → Чакры

| Лока | Плотность | Чакра | Принцип |
|------|-----------|-------|---------|
| [[../entities/lokas/bhur-loka|Бхур-лока]] | +1 | Муладхара | Выживание, тело, земля |
| [[../entities/lokas/bhuvar-loka|Бхувар-лока]] | +2 | Свадхистана | Эмоции, астрал |
| [[../entities/lokas/svar-loka|Свар-лока]] | +3 | Манипура | Воля, действие |
| [[../entities/lokas/mahar-loka|Махар-лока]] | +4 | Анахата | Равновесие, любовь |
| [[../entities/lokas/jana-loka|Джана-лока]] | +5 | Вишудха | Знание, речь |
| [[../entities/lokas/tapa-loka|Тапа-лока]] | +6 | Аджна | Концентрация, видение |
| [[../entities/lokas/satya-loka|Сатья-лока]] | +7 | Сахасрара | Истина, единство |

## В игре
- `vedic-cosmos.html` — 3D-визуализация всех 14 лок
- `initiation-corridor.html` — прохождение через локи в коридоре инициации
- `js/canon-module.js` → `loadLocas()`, `loadChakras()`

## Открытые вопросы
- Как 8-я и 9-я чакры (Монада, Абсолют) соотносятся с локами?
- Есть ли в игре механика прокачки чакр через локи?

# Связи между модулями

Создано: 2026-05-24

## Цепочки зависимостей

### Основная цепочка
```
core-module.js → state-module.js → ui-module.js → canon-module.js
                                                    ↓
                    lobby-module.js ← game-module.js ← universe-module.js
                         ↓
                    tigel-module.js
```

### Дополнительные модули
- `matrixSwitcher.js` → используется в `menu-module.js`
- `daimon-module.js` → `milost-module.js` → интеграция в Тигель
- `oracle.js` → `oraclePromptBuilder.js`
- `hints-module.js` → `i18n-module.js` → `i18n.js`

### Данные → Модули
- `data/agents.json` → `canon-module.js` → `universe-module.js`
- `data/matrices.json` → `canon-module.js` → `matrixSwitcher.js`
- `data/locas.json` → `canon-module.js` → `vedic-cosmos.html`
- `data/chakras.json` → `canon-module.js`
- `data/agent_matrix_map.json` → `canon-module.js`
- `data/daimon-forms.json` → `daimon-module.js`
- `data/milost-sources.json` → `milost-module.js`

## Экран → Модули
- `index.html` → `lobby-module.js`, `menu-module.js`, `i18n.js`, `matrixSwitcher.js`
- `tigel.html` → `tigel-module.js`, `state-module.js`, `ui-module.js`
- `universe.html` → `universe-module.js`, `canon-module.js`
- `initiation-space.html` → встроенная логика (Canvas)
