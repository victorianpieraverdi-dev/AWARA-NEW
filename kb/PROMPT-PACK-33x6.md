# AWARA · УСТРОЙСТВО ПАКЕТА ПРОМТОВ 33×6

«Устройство» (генеративный каркас) для создания **33 матриц × 6 ступеней = 198 образов** — карт-вселенных, в которых игрок пребывает на своей ступени постижения данной матрицы. Сюда же — каркас музыкального сопровождения по тем же ступеням.

Цель: чтобы любой нагенерированный промт (картинка или трек) **гарантированно ложился на лор**: канон Меры, палитру/мотивы линзы и логику Восхождения.

Связь с игрой:
- Ступень линзы считается от освоенности матрицы: `STATE.lenses[<key>].uses` → 1–6 (пороги: 0→1, 2→2, 5→3, 9→4, 14→5, 20+→6). Модуль `awara-ascension.js`.
- Промт карты дня уже подмешивает строку ступени (`window.lensAscensionPrompt`). Когда заполнишь JSON (см. §5), игру можно будет переключить на готовые промты вместо формулы.
- Имена файлов (договорённость):
  - картинки: `exports/generated_cards/lens_levels/{slug}_l{N}.webp`
  - музыка: `exports/audio/lens_levels/{slug}_l{N}.mp3`

---

## §1. Канон: что есть «ступень»

Канон: **Восход = КАЧЕСТВО Меры. Мера 1 = физика, 2 = искажение, 3 = рай на земле** (мягкая трактовка). Шесть ступеней — это не шесть «мер», а плавный подъём качества света *внутри* одной матрицы: от плотной физики через рассеивание искажения к раю-на-земле и далее к божественному апофеозу этой вселенной.

Одна и та же матрица (например, ведическая) на ступени 1 — тусклый бытовой храм в сумерках; на ступени 6 — золотой рай-космос, где игрок сам стал божеством этой традиции. Меняется НЕ сюжетная символика матрицы (мотивы те же), а **качество света, материи, детализации и музыки**.

## §2. Лестница 6 ступеней (универсальная, красится под матрицу)

Для каждой ступени: суть · свет/материя · эмоция · художественная обработка (EN-теги для генератора) · музыкальное направление.

**L1 — ФИЗИКА** (Мера 1)
- Суть: плотная материя, приземлённость, обыденность.
- Свет/материя: тусклый, рассеянный, тяжёлая фактура, пыль.
- Эмоция: усталость, поиск опоры, дремота духа.
- EN: `dim muted light, heavy earthbound matter, coarse grainy texture, mundane, low saturation, overcast, dense shadows`.
- Музыка: один простой инструмент традиции, тихо, медленный бытовой пульс, почти тишина.

**L2 — ПРОБУЖДЕНИЕ** (выход из искажения, Мера 2)
- Суть: первые искры, туман искажения начинает рассеиваться.
- Свет/материя: холодный рассветный луч пробивается, дымка тает.
- Эмоция: тревожный поиск, надежда, зов.
- EN: `cool dawn light breaking through haze, first sparks, mist dissolving, awakening, restless searching glow`.
- Музыка: нарастающий дрон, далёкий зов, вступает второй голос.

**L3 — ГАРМОНИЯ** (начало рая на земле, Мера 3)
- Суть: ясность, благословенность, мир.
- Свет/материя: тёплый ровный свет, чистые формы, баланс.
- Эмоция: покой, благодарность, «дома».
- EN: `warm balanced light, clear blessed atmosphere, harmonious, serene, gentle bloom, paradise-on-earth calm`.
- Музыка: гармоничная мелодия, дыхание, мягкая ритмика, согласие голосов.

**L4 — СИЯНИЕ**
- Суть: проступает священная геометрия, святость формы.
- Свет/материя: светящиеся контуры, нимбы, прозрачность.
- Эмоция: трепет, открытость, восторг узнавания.
- EN: `radiant luminous halos, sacred geometry emerging, glowing contours, translucent light, reverent awe, fine gold linework`.
- Музыка: хор и обертоны, расширение пространства, колокол/чаша.

**L5 — ВОСХОД**
- Суть: восходящее преображение, мир будто соткан из света.
- Свет/материя: золотой божественный свет, левитация, лучи.
- Эмоция: подъём, ликование, преодоление.
- EN: `golden divine light, ascending transcendence, world woven of light, rays of glory, levitation, triumphant radiance`.
- Музыка: эпический подъём, литавры и полный хор, кульминация.

**L6 — БОЖЕСТВЕННОЕ**
- Суть: рай этой вселенной; игрок — божество в собственном космосе матрицы.
- Свет/материя: бесконечное сияние, безграничный сакральный простор.
- Эмоция: единство, блаженство, полнота.
- EN: `utterly divine, heavenly realm of infinite radiance, godlike presence, boundless sacred cosmos, transcendent perfection, masterpiece`.
- Музыка: космический апофеоз, бесконечный хор, неземные обертоны, свет-как-звук.

## §3. Формула промта (слоты)

Собирай промт из слотов в этом порядке (EN для генераторов картинок):

```
{HERO}, deity-protagonist of the day, embodying {MATRIX_ART_BASE};
{LEVEL_TREATMENT (из §2)};
dominant palette {PALETTE};
sacred motifs: {MOTIFS};
mood: {MATRIX_TONE} + {DAY_FEELING};
portrait 3:4, single centered figure, symbolic, highly detailed,
{LEVEL_QUALITY_TAGS (из §2)}.
```

Где:
- `{HERO}` — герой дня (можно «a luminous human soul» или конкретный образ игрока).
- `{MATRIX_ART_BASE}`, `{PALETTE}`, `{MOTIFS}`, `{MATRIX_TONE}` — из §4 (якоря линзы).
- `{LEVEL_*}` — из §2 (ступень).
- `{DAY_FEELING}` — то, что игрок реально прожил (опционально, для карты дня).

Принцип силы промта: мотивы матрицы НЕ меняются от ступени; меняются свет, материя, детализация, эпичность. Чем выше ступень — тем больше «золота, света, сакральной геометрии, мастерства».

## §5. Схема выходного файла (для игры)

Когда нагенеришь — складывай в `data/lens_level_prompts.json`:

```json
{
  "vedic": {
    "1": { "prompt": "...EN prompt...", "music": "...муз. бриф...", "img": "lens_levels/vedic_l1.webp", "audio": "lens_levels/vedic_l1.mp3" },
    "2": { "prompt": "...", "music": "...", "img": "lens_levels/vedic_l2.webp", "audio": "lens_levels/vedic_l2.mp3" }
    // ... до "6"
  }
  // ... все 33 slug
}
```

Поля `img`/`audio` — опциональны (заполняешь по мере генерации ассетов). Когда файл появится, фронт можно научить читать `prompt` напрямую и подменять им формулу `lensAscensionPrompt`, а `img`/`audio` — подхватывать в карту/орб и в плеер ступени.

## §7. Музыка (каркас, по очереди)

Та же лестница §2 задаёт музыкальную дугу. Базовый инструмент/лад — от традиции матрицы (например, vedic: ситар/танпура/мантра; norse: костяная флейта/варган/мужской хор; egyptian: арфа/систр; cosmic_galactic: эмбиент-пэды/хор-обертоны). Ступени:
- L1: один инструмент, тихо, медленно, почти тишина.
- L2: дрон + второй голос, нарастание, зов.
- L3: гармоничная мелодия, дыхание, согласие.
- L4: хор/обертоны, колокол-чаша, расширение.
- L5: эпический подъём, перкуссия+полный хор, кульминация.
- L6: космический апофеоз, бесконечный хор, свет-как-звук.

Имена: `exports/audio/lens_levels/{slug}_l{N}.mp3`. В JSON (§5) — поле `audio` + краткий `music`-бриф.
