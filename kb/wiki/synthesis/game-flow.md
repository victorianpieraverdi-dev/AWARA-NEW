# Игровой поток (Game Flow)

Создано: 2026-05-24

## Путь игрока

```
ВХОД (Alpha Gate)
  ↓
ЛОББИ (Золотое Яйцо)
  ├─ ИГРА
  │   ├─ Матрицы (выбор восприятия)
  │   ├─ Карты (глобус, ведический космос, яйцо)
  │   ├─ Уровни
  │   │   ├─ Коридор Инициации (13 шагов)
  │   │   ├─ Пространство Души (3 сферы)
  │   │   ├─ Земля Игрока (Васту)
  │   │   ├─ Вселенная (агенты на орбитах)
  │   │   └─ Супер Игра (Мироздание)
  │   └─ Тигель (лог дня)
  ├─ ХРОНИКА
  │   ├─ Смысл дня
  │   ├─ Паспорт Души
  │   ├─ Колода Карт
  │   └─ Натальная Карта
  ├─ ОБМЕН
  │   ├─ Светкоин
  │   ├─ Милость
  │   ├─ Даймон
  │   └─ Артефакты
  └─ ОБЩЕНИЕ
      ├─ Оракул
      ├─ Чат Матрицы
      └─ Фестивали
```

## Ежедневный цикл
1. Зайти → Ключ Дня (агент + матрица + стихия)
2. Тигель → записать день → свет
3. Оракул → совет
4. Паспорт → прогресс

## Слои реальности
1. Аура человека (чакры, меры)
2. Регион / локация
3. Додекаэдр Земли (62 узла)
4. Чатуралока
5. Брахманда (14 лок)
6. Космические силы
7. Эпилог / возврат
