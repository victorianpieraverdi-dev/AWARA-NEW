# Иерархия карт и рарностей · AWARA

Создано: 2026-05-28 · Источник: анализ `data/*.json` в master (ff/awara-game) + дизайн-разговор с Павлом

Расширяет [[wiki/mechanics/card-system|Карточную систему]]. Привязка к [[wiki/synthesis/vision-update-2026-05-28|Vision-апдейту]] (T-612f «карточная игра: иерархия по свету + место в мироздании»).

## Главное открытие (из данных, не из догадок)

**Рарность уже привязана к АГЕНТУ (вертикаль сознания), а не к культуре.** Проверено по `card_prompts.json` (693 = 21 агент × 33 матрицы): распределение рарности для каждого агента одинаково во всех 33 культурах. Культура (матрица) — горизонталь, агент — вертикаль. Это подтверждает дизайн «матрица = линза, агент = высота».

Группы агентов по среднему рангу рарности:
| Ранг | Агенты | Доминанта |
|---|---|---|
| 0.39 | ananda, dharma, iskra, jnana, karma, prema, shanti | common |
| 0.94 | agni, prithvi, tejas, varuna, vayu (стихийные) | uncommon |
| 1.58 | lakshmi, parvati, sarasvati, shakti | uncommon→epic |
| 2.15 | shiva, svet_ra, vishnu | epic |
| 2.94 | akasha, brahma | epic→legendary |

Вывод для дизайна: **рарность карты = высота её агента на оси сознания.** Это и есть «место в мироздании» = рарность. Естественно гейтится через `totalLight` (см. [[wiki/concepts/light-economy|Экономика света]]).

## Шкала рарности (6 тиров, факт из данных)

`common → uncommon → rare → epic → legendary → mythic`

(в `card-system.md` раньше было 5 — пропущен **uncommon**, исправлено)

## Предлагаемая иерархия (две независимые оси)

Из разговора с Павлом — НЕ утверждено, на ревью:

**Ось 1 — Вертикаль (рарность = высота сознания).** Уже зашита в данные через агентов. Редкая карта = высокий агент (akasha/brahma) → открывается при большом `totalLight`. Рарность и прогресс — одна ось (когерентно с «свет = гейт»).

**Ось 2 — Горизонталь (культура = линза, НЕ ранг).** 33 матрицы равноправны. «Иерархия матриц по свету» в лоб не ложится (нельзя «египет светлее славян»). Вместо ранга культур — **глубина внутри культуры** (ключи → фрагменты → знания → карточки, см. `quest-cultural-keys.json`).

**Мифо-слой (+мифология 33 культур):** рарность мифокарты = её высота **в космологии своей культуры** (локальная вертикаль пантеона), а не сравнение культур между собой.

**Ширина (сколько матриц активно одновременно):** гейтится светом. Старт = 3 матрицы (свободный выбор), прогресс → +2. См. [[wiki/concepts/matrix-choice|Выбор матрицы]].
