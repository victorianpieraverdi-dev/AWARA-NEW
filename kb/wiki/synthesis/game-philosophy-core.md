# Философское ядро AWARA

Создано: 2026-05-26 · Источник: интервью Pavel + Claude

## 7 Принципов

### 1. Игра = Дневник Совести
Это не развлечение. Это журнал. Игрок записывает действия → игра отражает уровень развития → игрок растёт.
**Сознание игры** — это прошедший игру игрок, который ведёт текущего.
Ключ: честность, открытость, совесть.

### 2. Прогрессивное раскрытие (Grayscale → Color)
- Игрок начинает в **чёрно-белом** интерфейсе
- По мере накопления света: цвета, краски, матрицы, голоса, возможности
- Чем больше света — тем ярче, глубже, красивее игра
- **Кнопка Создателя** (Ctrl+Shift+C) — режим быстрого пролистывания уровней для теста

### 3. От Ума → к Чувствам
- **Первый этап:** ментальные конструкции, намерения, планы (пространство Ума)
- **Второй этап:** чувствительность, эмоции, эмпатия (пространство Души)
- **Третий этап:** интуиция, сиддхи, взаимодействие с тонким планом
- **Четвёртый этап:** планетарный Логос, космическое сознание
- Игра ведёт от рационального к интуитивному восприятию

### 4. Матрицы = Культурные Линзы
- **Ведическое ядро** (21 агент) — неизменно
- Матрицы — линзы восприятия, через которые игрок взаимодействует
- Игрок выбирает сам, но ограниченно:
  - Сначала 3 основные матрицы
  - Потом открываются новые (по свету)
  - Можно менять если не подходит
- Свобода выбора важнее навязывания

### 5. Свет — Универсальный Обмен
- Свет копится из эфира (5 стихий: земля, вода, огонь, воздух, эфир)
- Эфир → вибрация → свет
- Свет можно тратить на ВСЁ:
  - Развитие интерфейса
  - Открытие матриц
  - Голос Оракула (более сильные AI-модели)
  - Глубину игры
  - Визуальное наполнение
- **Формула:** стихии → эфир → вибрация → свет → прогресс

### 6. Живые Переходы
- Между экранами — микро-события (карта дня, фрагмент лора)
- В будущем: интеграция с Telegram, Twitter, Instagram
- Игрок не может переходить между ВСЕМИ экранами сразу — сначала наполняет игру
- Игра живая — она реагирует на действия игрока

### 7. Онбординг: Показать Финал → Привести к Началу
- Игрок видит презентацию максимального опыта
- Его возвращают на первый экран
- Он начинает путь с первого шага
- Постепенное заполнение и рост
