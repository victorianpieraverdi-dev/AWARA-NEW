## 1. Мультивселенная и космология (каркас мира)
**Идеи:** мультивселенные + отдельная папка под каждую из 33 Вселенных (космос, герои, отношения, история внутри); кнопка, меняющая весь сайт — «Супер Ап» как в мультивселенной Паука; выгрузка целых локаций, игра на местностях; «на земле от поля пустоты до звёздного храма», от ничто до создания звёзд и кораблей, путешествия по мирам.

**Куда ложится:** расширение экрана **Макрокосм** (#195 уже даёт 22 пузыря-вселенные + Сферу Золотого Века). 33 матрицы уже существуют как данные.

**Как внедрять:**
- Единая схема данных вселенной: `data/universes/<id>/universe.json` (космос, герои, связи, история, палитра/правила). Сейчас `cosmic_forces.json`, `vedic_loka.json`, `chatura_loka.json` — это де-факто первые вселенные; обобщить их в один формат.
- «Супер Ап кнопка» = смена активной вселенной → перекраска всего сайта через CSS-переменные + конфиг вселенной (глобальный UniverseContext: меняет тему/токены, подгружает ассеты).
- Пузыри в #195 делаем кликабельными → вход во вселенную (загрузка её локации).

## 2. Визуальная эволюция (ч/б → 3D → стиль)
**Идеи:** сначала всё плоское ч/б → потом 3D → далее; 4 кнопки (2D ЧБ / ЧБ движение / 3D+звук+цвет+глубина / стиль+управление объектами и законами, «сознание-пластилин», окна местами); арты под окна и элементы стиля + части матрицы прорисовываются по процентам; стиль-референс «Донат»; рисунок = вибрация.

**Куда ложится:** ПРЯМО на существующую механику `grayscale-to-color`. Это центральная визуальная метафора прогрессии.

**Как внедрять:**
- 4 ступени = глобальный уровень проявленности `renderTier 0..3`, завязанный на прогресс/свет:
  - 0 — 2D ЧБ (плоские силуэты)
  - 1 — ЧБ + движение (анимация, параллакс)
  - 2 — 3D + цвет + звук + глубина
  - 3 — управление объектами/законами (sandbox, «сознание-пластилин», перестановка окон)
- «Части матрицы по процентам» = заполнение арта окна пропорционально прогрессу (mask reveal на том же шейдере ч/б→цвет).

---

## 3. Даймон (спутник и аватар игрока)
**Идеи:** аватары даймона (несколько форм, уровни); артефакты и силы даймона; мелодии/песни даймона 🐕; может ли натальная карта рассчитываться через матрицу или с элементами для даймона; варианты, когда человек не даёт о себе ничего.

**Куда ложится:** существующая система даймона (`daimon-manager.js`, `daimon-card.js`, `daimons/`).

**Как внедрять:**
- Эволюция аватара даймона = формы по уровням, привязать к мандале души / `renderTier`.
- **Онбординг без данных:** даймон формируется эвристически (Voice of Truth) из минимальных выборов — НЕ требует натальных данных. Натальная карта = опциональный «точный» режим.
- Силы/артефакты даймона = инвентарь, влияющий на карту дня и доступы.
- Мелодии = аудиослой (отдельный backlog).

---

## 4. Карты, настолка и ключи (ядро геймплея)
**Идеи:** настолка отражает всю игру, карточки = карточки; карточки открывают доп/скрытые функции и доступ к генераторам смысла ВидьЯ; КЛЮЧ-КАРТА — активатор энергий, сочетания меняют игру на время; обмен элементами/стихиями, доп карты; редкие — только вместе (кооп-крафт); генерации = фрагменты ключа/силы; карточка дня из качеств+атрибутов+даймон; формат карточки под игру + профиль игрока.

**Куда ложится:** существующая `card-system`.

**Как внедрять:**
- Единая модель карты: `{качества, стихия, редкость, эффект-ключа, история}`.
- **Карточка дня** = ежедневный генератор из текущего `PlayerState` (качества/атрибуты) + даймон. Это и есть ядро Phase 1 daily loop.
- Ключ-карты = временные модификаторы правил (бафы/доступы); сочетания → комбо-эффекты.
- Генерации (арты/3D) выдают фрагменты ключей → стимул генерировать.
- Обмен / «редкие только вместе» = кооп-соц-слой (поздняя фаза).

---

## 5. Тигель и алхимия (трансформация форм)
**Идеи:** тигель из форм и бутылей, наливать воду, логика распределения, что трансформируют сосуды, усложнить восприятие/ум; при достатке восприятия форма высвобождается, осознаёт себя, меняет форму, тянет остальных; скрещивание предметов/качеств/энергий (водяная рудра света, огонь земли трёх солнц); манас — 4D-принтер, творить материальный манас; артефакты и их истории.

**Куда ложится:** существующий **Тигель** (`cauldronEngine.js`, `tigel-app.html`) — сейчас в активной работе.

**Как внедрять:**
- Тигель = крафт-движок: сосуды + жидкости/энергии → правила смешивания → новый артефакт/качество.
- «Самоосознание формы» = порог восприятия: при накоплении параметра форма эволюционирует (эмёрджент-механика трансмутации).
- Манас-4D-принтер = выходной модуль тигля: материализует результат в артефакт/объект (3D-генерация — позже).

## 6. Мандала души (мета-прогрессия сознания)
**Идеи:** мандала души от тёмного хаоса/демона до Бога через порядок и свет; тёмные глухие сферы → свет звёзд, золото, камни, живые сущности из матриц; эволюция, милость, благословение, осознанность, выбор; уровни «мост-туннель-сфера-туннель-сфера», проход через меры/чакры; следующий уровень включает предыдущий + мудрость/сила/свет + особые артефакты; конечная — высший мир, эволюция Логоса, 64 точки кристалла, массаж/дыхание, переход в новое измерение.

**Куда ложится:** глобальная прогрессия игрока (`core/types.ts` → `PlayerState`, `CHAKRA_NAMES` 9), связка узлов 0→7.

**Как внедрять:**
- Мандала души = визуализация `PlayerState`: сферы/чакры заполняются светом по мере прогресса. Рифмуется с `renderTier` и ч/б→цвет.
- Структура «туннель-сфера» = переходы между узлами (0→7) и между `renderTier`; каждый уровень накопительно включает предыдущий.
- 64 точки кристалла / дыхание = финальная практика-ритуал (поздняя фаза, возможно вне Phase 1).

---

## 7. Свет, прокачка, экономика, монетизация
**Идеи:** золотые рамки/аватарки/элементы дизайна — покупать и качать из разных вселенных, строить своё пространство; всё прокачивать и запускать; кнопки, запускающие глубину «за свет» или «за то, что ты первый»; свет открывает доступ ко всем функциям; истории прохождения, заслуги.

**Куда ложится:** новый слой прогрессии/экономики поверх Phase 1.

**Как внедрять:**
- **«Свет» = основная валюта прогресса** (зарабатывается в daily loop / квестах) → открывает функции и глубину (`renderTier`, окна, тигель).
- Косметика (золотые рамки, элементы из вселенных) = персонализация пространства; «качать из вселенных» = коллекция.
- Заслуги / истории прохождения = achievements + журнал.
- «Первый в игре» = early-bird бонусы.

---

## 8. Контент-движок и обучение (ВидьЯ)
**Идеи:** умные философские мысли согласно уровню и матрице, перевод всего + обучение; генераторы смысла ВидьЯ.

**Куда ложится:** Voice of Truth / контент-слой (`core/ai/VoiceOfTruth.ts`).

**Как внедрять:**
- ВидьЯ = библиотека смыслов/афоризмов, отдаваемая по уровню+матрице игрока. Phase 1 — заготовленный корпус + эвристический подбор; Phase 2 — LLM.
- «Перевод всего» = у каждого игрового объекта есть философский слой-толкование.

---

## 9. Мета: смысл, внешка, рост
**Идеи:** сайт игры, соцсети, инвесторы; миссия/предназначение/Созидание; пробуждение Логоса = Игра; Родовая Сила — новый уровень: Деньги, Власть, Контроль/Управление, Магия.

**Куда ложится:** продуктовый/нарративный слой (частично вне движка).

**Как внедрять:**
- North Star: «Пробуждение Логоса / Созидание» — верхнеуровневый нарративный стержень (зафиксировать в концептах).
- Сайт/соцсети/инвесторы = маркетинг-трек (отдельно от кода).
- «Родовая Сила (Деньги/Власть/Магия)» = поздний контентный уровень/ветка прогрессии.

## Карта внедрения по фазам

### Сейчас / Phase 1 (дёшево + усиливает ядро)
- `renderTier 0..3` как глобальный флаг проявленности (сист. 2) — связать с ч/б→цвет.
- Карточка дня из `PlayerState` + даймон (сист. 4) — ядро daily loop.
- Онбординг без данных через эвристику (сист. 3).
- Базовая модель карты `{качества, стихия, редкость, эффект, история}` (сист. 4).
- Зафиксировать North Star: Пробуждение Логоса / Созидание (сист. 9).

### Средняя фаза (расширение)
- Схема `universe.json` + кликабельные пузыри → вход (сист. 1).
- Формы/уровни аватара даймона (сист. 3).
- Ключ-карты как модификаторы правил + комбо (сист. 4).
- Тигель: эмёрджентное самоосознание формы (сист. 5).
- Мандала души как UI прогресса (сист. 6).
- «Свет» как валюта доступа (сист. 7).
- Корпус ВидьЯ по уровням (сист. 8).

### Поздняя фаза (генерация / LLM / соц)
- «Супер Ап» — полная смена вселенной и темы сайта (сист. 1).
- Натальный расчёт даймона (сист. 3).
- Обмен картами, «редкие только вместе», кооп (сист. 4).
- Манас-4D-принтер с 3D-генерацией (сист. 5).
- 64 точки кристалла / финальный переход (сист. 6).
- Магазин косметики, монетизация (сист. 7).
- LLM-генерация ВидьЯ (сист. 8).
- Родовая Сила: Деньги/Власть/Магия (сист. 9).

## Сквозная связь (почему это одна игра, а не список)
**Свет** (валюта) зарабатывается в daily loop (**карточка дня**) → повышает **renderTier** (ч/б→3D) и заполняет **мандалу души** → открывает **окна/вселенные** и **тигель** → тигель/генерации дают **артефакты и фрагменты ключей** → **ключ-карты** меняют правила → глубже в **узлы 0→7** → финальный **переход (эволюция Логоса)**.
