# Архитектура экранов AWARA 2.0

Зафиксировано: 2026-06-15

## Васту-крест (главная навигация)

```
                  [5 ДУША]
                     ↑
[4 ДАЙМОН] ← ← [2 ЯЙЦО] → → [6 НАСТОЛКА]
                     ↓
                [3 МАТРИЦЫ]

Вход: 0 (пустота/искра) → 1 (Вселенная) → 2 (центр)
```

Навигация: кнопки ←↑↓→ по краям + свайп. Переходы плавные (окно в окно, не слайды).

## Ноды подробно

### 0 — Вход (Entry)
- Тёмный космос, мистический круг
- При наведении: пульсирует искра Света внутри
- Клик → переход в 1

### 1 — Вселенная
- Торус / персональная вселенная игрока (рандомный цвет/форма при создании)
- Свайп влево-вправо → другие вселенные других игроков (1.3)
- 1.4 — 14 Лок, 3D ведическая космология
- 1.4.4 — Космические силы (агенты)
- 1.1 — Чатур-лока (4-слойная сфера, человек в центре)
- 1.2 — Человек: два вида — 9-мерный И 7-чакрный
  - 1.22 — Дерево (вправо от человека)
  - 1.23 — Кристалл Души / Меркаба (влево от человека)

### 2 — Центральное Яйцо (Лобби)
- Яйцо разделено на 4 части (4 направления)
- Внутри: 7 чакр по вертикали (возможно 9)
- **Дополнительные элементы вне яйца:**
  - Левый верх: уровень игрока + дашборд
  - Правый верх: меню (активирует 4 окна внутри яйца)
  - Правый низ: чат (гильдия, другие игроки)
  - Левый низ: мини-игры
- Трансформируется со Светом игрока

### 3 — Матрицы / Ключи / Информация (Запад)
- 33 матрицы культур + ключи
- Что сегодня: звёзды, астрология, энергия дня
- 3.1 — Матрицы и ключи: философия каждой, нарабатываются через прохождение
- 3.2 — Коллекция карт (копятся из пройденных матриц)
- 3.3 — Индивидуальная духовная сущность игрока (формируется из прошедших матриц)

### 4 — Даймон (Юг)
- Личный спутник/существо, генерируется из даты рождения
- Эволюционирует, имеет уровни, растёт с игроком
- 4.1 — Квесты через Даймона
- 4.2 — Скрещивание / сражения с Даймонами других игроков (Покемон-механика)
- Резонирует с записями в Тигеле

### 5 — Душа (Восток) — самый развёрнутый раздел
- 3 основные сферы + 7 подсфер (прошлое/настоящее/будущее + физ/псих/энерг/ум/каналы)
- Голос Совести (ИИ) распределяет энергию дня по всем подсистемам

**Подноды:**
| Нода | Содержание |
|---|---|
| 5.1 | Локация (интерактивная карта где игрок физически) |
| 5.2 | Тонкий план (видения, мысли, места силы) |
| 5.3 | Карта мира (регион/страна, более широко) |
| 5.4 | Пространство Ума / Тело (меридианы, чакры тела) |
| 5.5 | Космос игрока (выше Храма, звёздные силы) |
| 5.6 | Другие вселенные (вселенные других игроков) |
| 5.8 | Световые структуры (за пределами богов, новое созидание) |
| 5.9 | Звёздный Храм (Вастушастра, стороны света, стихии, агенты) |
| 5.10 | Квесты по конкретным агентам |
| 5.11 | Карта Храмов (все построенные, до 33+21) |
| 5.12 | Содействие — совместное строительство Храмов |
| 5.13 | Совместные события с другими игроками |
| 5.14 | Пространство совместных действий |
| 5.9 / 5.12 | Переход в Космос игрока (через Храм) |
| Календарь | Кнопка в 5 — планировщик и дата |

**Над 5:** RA БАТЯ / Храм Бога (финальный уровень)

### 6 — Настолка / Карта Мироздания (Север)
- Концентрические кольца от периферии к центру (RA)
- Кольца: 7 чакр → 9 измерений → 14 Лок → агенты → знаки Зодиака → …
- 10 уровней: от ума (периферия) до Источника (центр)
- 6.1 — Додекаэдр Планетарного Логоса (62 точки силы)
- 6.2 — Связи с другими игроками, карты других
- 6.3 — Световой Маркетплейс (услуги, обмен)
- 6.4 — Инвестиции / Банк

### 7 — Фестивальное пространство (концепция)
- Физические фестивали, движение по сторонам света
- Театр кукол / Даймон-архетипы для детей
- Реальные артефакты из стихий
- Связано с реальной геолокацией

## Переходы между нодами

- 0 → 1 → 2: последовательный вход в игру
- 2 → 3/4/5/6: Васту-крест (влево/вниз/вверх/вправо)
- 5 → 5.5 → 6: Душа → Космос → Мироздание (вертикальный рост)
- Все переходы: плавные, window-style (не слайды)
- Мобайл: свайп жестами
