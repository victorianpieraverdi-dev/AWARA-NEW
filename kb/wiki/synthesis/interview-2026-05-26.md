# Интервью: Дизайн Игры · 2026-05-26

Создано: 2026-05-26 · Участники: Pavel + Claude

## Затронутые темы

- Петля игрока и момент прогресса
- Свет как универсальная валюта обмена
- Пустота между экранами → живые переходы
- Матрицы: выбор vs судьба
- Первый опыт входа в игру

## Ключевые решения

1. **Игра = Дневник Совести** — не развлечение, а журнал развития
2. **Grayscale → Color** — интерфейс раскрывается по мере роста света
3. **От Ума к Чувствам** — этапность развития игрока
4. **Матрицы — выбор игрока** — свобода выбора, ограниченное количество на старте
5. **Свет тратится на ВСЁ** — интерфейс, AI, матрицы, цвета
6. **Живые переходы** — микро-события между экранами
7. **Онбординг: показать финал → привести к началу**

## Решения по матрицам
- Игрок выбирает сам (не навязываются)
- Старт: 3 матрицы
- Открытие новых по свету или по запросу
- Возможность сменить матрицу если не подходит
- Матрица = культурная линза на ведическое ядро

## Решения по свету
- Свет конвертируется в развитие интерфейса
- Свет открывает более сильные AI-модели для Оракула
- Свет открывает цвета и краски игры
- Свет = эфир из 5 стихий = вибрация = прогресс
