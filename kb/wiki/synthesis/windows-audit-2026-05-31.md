# Ревизия окон AWARA + план Режима создателя

> Точка для хэндофа: если упрёмся в лимит — отсюда продолжаем «структурирование окон и Режим создателя».

## 1. Контекст
За ~2 месяца накопилось 40+ `.html` окон в `C:\AWARA`. Цель: навести порядок, спрятать большую часть за **Режим создателя**, оставить чистый тестируемый хребет для нескольких тестеров.

## 2. Решения пользователя (31.05)
- **Что видят тестеры:** широкий core — все три хороших группы (минимал-вход + дневной цикл + Даймон/сферы Души). **Карты подключаются постепенно.**
- **Старые дубли** (`ref-initiation-v258.html`, `index-pre-egg.html`, `universe-old.html`): **пока оставить как есть** (файлы не трогаем, не удаляем).

- `index.html` — Лобби «Золотое Яйцо»

- `index-pre-egg.html` — лобби до «Золотого Яйца»

## 6. Архитектурная находка (важно для реализации)
Два параллельных мира:
- **Легаси:** `index.html` (1.16 МБ) + `init.js` / `game-module.js` на глобалах (`window.openGameSpaces()`, глобальный `state`, `YugaSystem`).
- **Чистый модульный:** `js/menu-module.js` — реестр всех окон (4 категории: ИГРА / ХРОНИКА / ОБМЕН / ОБЩЕНИЕ), у каждого пункта `minLight`, есть `getAccessibleItems(light)` / `getLockedItems(light)`.

`menu-module.js` — естественный центральный «рубильник» для гейтинга. **Нужно подтвердить, рендерит ли живое лобби именно `MENU` из этого модуля, или инлайновый легаси-код в `index.html`.** От этого зависит подход.

## 7. План Режима создателя
1. В `menu-module.js` добавить хелперы: `isCreatorMode()` / `setCreatorMode(on)` / `toggleCreatorMode()` (localStorage ключ `awara_creator`).
2. Ввести у пунктов меню флаг `creatorOnly` и/или градуированный `minLight`.
3. `getAccessibleItems()`: в Режиме создателя показывать всё; иначе скрывать `creatorOnly` и пункты выше текущего света.
4. Тостер-core = `minLight: 0`; карты/глубокие уровни — градуированные пороги (3000/7000/…), завязанные на существующие `ACCESS_LEVELS` и систему Юг.
5. Тоггл Режима создателя: `Ctrl+Shift+C` + видимый бейдж (по game-philosophy-core такой хоткей уже упоминался — проверить).
6. Если живое лобби НЕ читает `MENU` — добавить тонкий standalone-гард `js/creator-mode.js`, подключаемый в `index.html`, который прячет ссылки на creator-only окна и ставит редирект при прямом заходе.

## 8. Следующий шаг (открытый вопрос)
