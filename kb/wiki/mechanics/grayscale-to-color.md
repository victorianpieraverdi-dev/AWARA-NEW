# Grayscale → Color: Прогрессивное раскрытие интерфейса

Создано: 2026-05-26 · Источник: интервью Pavel

## Концепция

Игрок начинает в чёрно-белом, матовом интерфейсе.
По мере накопления света открываются цвета, краски, возможности.

## Механика

### Уровень 0 (0-500 света)
- Ч/б интерфейс
- Минимальный набор экранов (только пространство Души + Тигель)
- Базовый Оракул (слабая модель)
- 3 матрицы

### Уровень 1 (500-2000)
- Появляются первые цвета (золотой, синий)
- Открывается пространство Сердца
- 5 матриц
- Даймон активируется

### Уровень 2 (2000-7000)
- Полная цветовая палитра
- Пространство Разума + Земля
- 10 матриц
- Оракул на средней модели

### Уровень 3 (7000-25000)
- Глубокие цвета, градиенты, свечения
- Вселенная, Храмы
- 20 матриц
- Оракул на сильной модели

### Уровень 4 (25000+)
- Полный спектр, сакральная геометрия, анимации
- Мироздание, все экраны
- Все 33 матрицы
- Максимальный Оракул

## Техническая реализация

- CSS-переменные для цветовой схемы
- `data-theme="level0"` → `data-theme="level4"` на body
- Фильтр `grayscale()` → постепенно убирается
- Новые цвета добавляются через CSS custom properties
- Кнопка Создателя: Ctrl+Shift+C — переключение уровней для теста
