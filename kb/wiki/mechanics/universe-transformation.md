# Механизм преобразования вселенной (мёртвая → живая)

Как ре-ролльная вселенная превращается в **живую игровую** под действием сил, лок, заслуг
игрока и (в будущем) артефактов-порталов.

## Конвейер преобразования

```
        ВХОДЫ (силы)                  СВОРАЧИВАНИЕ              ВЫХОД (что видно)
  +-----------------------+      +------------------+     +---------------------+
  | 1 базовый спектр       |      |                  |     | цвет ядра / звёзд    |
  |   (ре-ролл на старте)  |----->| base = спектр   |---->| яркость / пульсация  |
  | 2 активные локи        |      | + Σ влияний × вес |     | плотность / "объём"   |
  | 3 заслуги игрока       |----->|                  |---->| renderTier (T0->T5) |
  | 4 тигель дня          |      | слияние в ЦЕНТРЕ|     | подвселенная (чакры) |
  | 5 натальная карта      |----->| + КАЙМА по краям|     | порталы / мосты      |
  | 6 транзит дня/завтра |      | (различение      |     |   к др. вселенным     |
  | 7 связи (агент/мир/  |----->|  природы цветов)|---->| мультивселенная      |
  |   человек/бог света) |      |                  |     |                     |
  | 8 артефакты (будущее)|      +------------------+     +---------------------+
  +-----------------------+
```

## Силы и их эффект

| # | Сила | Что меняет | Статус |
|---|------|------------|--------|
| 1 | Базовый спектр (ре-ролл) | собственный цвет вселенной | работает |
| 2 | Активные локи | поток света локи поверх базы | 1 лока дня, налёт 0.3 |
| 3 | Заслуги игрока (свет/квесты) | глубина, яркость, renderTier | тир есть, не завязан |
| 4 | Тигель дня | сила вливания лок сегодня | план |
| 5 | Натальная карта | врождённые веса лок | план |
| 6 | Транзит дня/завтра | какие локи активны и насколько | план (простой календарь) |
| 7 | Связи (агент/мир/человек/бог) | каждый добавляет свой поток | план |
| 8 | Артефакты → порталы | связки вселенных, мультивселенная | дальний горизонт (T5) |

## Лестница оживления (renderTier)

Ось «мёртвое → живое»:

- **T0** — плоско, статично, ч/б (мёртвая материя)
- **T1** — ч/б + движение (первое дыхание)
- **T2** — +свет/звук (чувства)
- **T3** — можно переставлять (воля)
- **T4** — можно создавать (творец) — здесь включается спектр лок
- **T5** — законы, порталы, связки вселенных → мультивселенная

## Формула оживления

```
живость = базовый_спектр + Σ(влияние_локи × сила)
сила = f(тигель_дня, натал, транзит, заслуги)
несколько влияний → слияние в центре + кайма по краям (видна природа каждого света)
```

Артефакты — это **«замороженные силы»**: предмет-портал несёт спектр чужой локи/вселенной
и, будучи активным, добавляет свой поток; на T5 — открывает мост в ту вселенную.

## Порядок реализации (снизу вверх)

1. Несколько лок + кайма (слияние в центре, следы по краям) — **текущий шаг**
2. Сила влияния от тигеля / натала / транзита / заслуг
3. Связи перекрашивают вселенную
4. Подвселенный уровень (человек: 14 чакр)
5. Оживление по renderTier T0->T5
6. Артефакты → порталы → связки вселенных → мультивселенная (T5)
