# Мифическая система

Создано: 2026-05-24

## Обзор

Мифическая система AWARA включает:

### Квестовые линии (5)
Из `mythic_quest_lines.json` (261 квест)

### Ветви развития (33)
Из `mythic_branches.json` — по одной на каждую матрицу

### Локации (231)
Из `mythic_locations.json` — места для квестов и путешествий

### Реликвии (231)
Из `mythic_relics.json` — артефакты силы

### Отношения (330)
Из `mythic_relationships.json` — связи между сущностями
