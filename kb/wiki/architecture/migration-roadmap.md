# AWARA · Дорожная карта переноса (Shell-first)

> Решение от 2026-06-07: **новая WebGL-оболочка с нуля + перенос старого контента по кускам.**
> Источник полного описания игры: `wiki/sources/game-design-voice-2026-06.md` (+ ручная карта-схема узлов 0–7).

## Принцип: два слоя

1. **Позвоночник (строим с нуля).** Единый движок, который задаёт «грандиозность» и цельность.
2. **Контент и логика (переносим, НЕ переписываем).** Годы наработок — переносим по узлам.

---

## Слой 1 — что строим с нуля (Shell)

- WebGL-движок на Three.js / R3F + GSAP-переходы (`core/`).
- `StateManager` — граф узлов 0–7, крест Васту, плавные переходы (spark / zoom / cross).
- **Динамическая Мерность** — при низком Свете UI плоский/монохромный, при высоком — 3D, объём, bloom (`DimensionalMaterial`).
- **Голос Истины** (`VoiceOfTruth`) — приём аудиодневника, разбор стихий, распределение «сока» по 7→9 чакрам, Бодхичитта-подсказки при энтропии.
- Дизайн-система: золото `#c9a84c` на `#000/#0a0a14`, шрифты Cinzel / Cormorant Garamond / JetBrains Mono.
- Сквозная двуязычность ru/en + обучающие подсказки на каждом шаге.

## Слой 2 — что переносим (золото, не трогаем заново)

- Астрономия **swisseph** (натальная карта, транзиты) → узлы 3 и 5.
- **33 матрицы** + философия каждой, **21 агент** → узел 3.
- `quest-engine`, `i18n`, данные, календарь-планёр.
- Готовый контент экранов (legacy `.html`) — пока работает через мост, переносится по фазам.

## Фазы

### Фаза 0 — Каркас (✅ готово)
- `core/` слой написан и закоммичен (`ca9515c`).
- Dev-стенд на Vite: узлы 0→1→2 нативно (Искра → Тор → Золотое Яйцо), мост к legacy 3–7.

### Фаза 1 — MVP «Ежедневная петля» (СЛЕДУЮЩИЙ ШАГ)
Сердце игры, на нём проверяется вся архитектура:
> вход (0→1→2) → запись аудиодневника дня → Голос Истины разбирает стихии → распределяет «сок» по 7 чакрам/сферам → Свет растёт → плоский UI становится объёмным.
- Подключить реальный LLM в `VoiceOfTruth` (сейчас эвристика-заглушка).
- Запись аудио в браузере (MediaRecorder) + транскрипция.
- Переспрашивание по пунктам (даймон / настолка / душа / локация / тонкий план).
- Сохранение состояния игрока (localStorage → потом Firebase).

### Фаза 2 — Лобби нативно (узел 2)
- Золотое Яйцо: 7→9 энергоцентров (вертикальный столб), крест Васти.
- Доп-меню вне яйца: слева-вверху уровень/дашборд, справа меню, справа-внизу чат/гильдия, слева-внизу мини-игры.
- Трансформация центрального окна по количеству Света (перенос окон, x/y/z-сферы).

### Фаза 3 — Душа / Тигель (узел 5)
Перенос подэкранов: 5.1 гео-карта мест силы · 5.2 тонкий план · 5.3 мир-карта · 5.4 меридианы тела (усинь) · 5.5 Космос игрока · 5.8 «65-й ген-ключ» · 5.9/5.12 строительство Звёздных храмов (Васту) · 5.10 квесты агентов · 5.11 город храмов (до 33) · 5.13/5.14 совместные события · календарь-планёр.

### Фаза 4 — Матрицы / Информация (узел 3)
- Перенос 33 матриц + философия, прохождение Искрой → 3.3 индивидуальная духовная сущность.
- 3.2 коллекция карт силы (сундуки/ключи) → используются в узле 6.
- Подключить swisseph: сегодняшний день, звёзды, натальная карта, транзиты, артефакты/кристаллы времени.

### Фаза 5 — Даймон (узел 4)
- Генерация из даты рождения (тень → дар → сиддхи), карма/наклонности.
- 4.1 питание + квесты, 3D-анимация из карточки.
- 4.2 скрещивание/битвы даймонов (как покемоны), влияние матриц/агентов.

### Фаза 6 — Световая настольная игра (узел 6) — самая сложная
- От периферии к центру (Ra): 7 центров · 9 измерений · 14 лок · агенты ± · сферы времени · 12 знаков · накшатры.
- Спектр прохождения: чёрный/серый/бурый/красный → оранжевый → жёлтый → зелёный → голубой → синий → фиолетовый → серебро → золото.
- 10 уровней сознания (ум → … → возврат в Исток).
- 6.1 додекаэдр планетарного логоса: 62 точки силы + 65-й элемент (созидание нового).
- 6.2 другие игроки и их карты · 6.4 банк/инвесторы · световой маркетплейс.

### Фаза 7 — Держава RA (узел 7)
- Тороид-хронограф (Круг ~10 000 лет, 4 секции: детство/Ренессанс/библиотека/любовь).
- Золотой купол (12 посвящённых) · поляна царя · Тёмный лес · Тёмная гора (отработка).
- Камни вечности / первородные камни созидания (трансцендентные энергии).
- Корни: фестиваль «Светоруссия», Камышловский район — мифологический исток.

### Сквозные системы (через все фазы)
- Мультиплеер: содействие, гильдии, совместные события/строительство.
- Мини-игры (доп-монеты), в т.ч. кукольный театр для детей (архетип через тень/дар ген-ключей).
- Энтропия: при простое Вселенная теряет светимость; игра предлагает позитивные выходы (Бодхичитта).
