## ✅ Сделано (журнал)

### Космос и сфера
- Глубокая тьма, редкие искры; живое звёздное поле с мерцанием.
- Параллакс-слои + лёгкая воронка (звёзды затягиваются к сфере).
- Тёмная сфера-пустота: почти чёрное «нутро», тонкий фреснель-ободок, золотой кант, разгорается при наведении.
- Idle-движение только через `transform`/`opacity` (без `background-position`) — починили лаги.

### Звезда / мандала (внешний ободок)
- Пробовали {7/3} гептаграмму → по решению вернули **9-лучевую Звезду Инглии** (3 пересекающихся треугольника, шаг 40°).
- Звезда теперь — **внешний ободок (obodok)**: внутренние пересечения приглушены (`smoothstep(0.34, 0.60, r)`), `R = 0.84`. Центр свободен под космогонию.
- Бегущая «сварочная» искра + мерцание по контуру звезды и ободка — на месте.

### Внутренняя космогония (разворачивается ПО ПРИБЛИЖЕНИЮ курсора, uh 0→1)
Теософская лесенка рождения Вселенной внутри сферы:
1. **Пралайя** — тихий светящийся круг (покой Абсолюта).
2. **Бинду** — первая точка творения в центре.
3. **Горизонталь** — диаметр: Дух / Материя.
4. **Крест** — добавляется вертикаль (Космический Разум, семя форм).
5. **Коловрат → спиральная галактика** — рукава креста закругляются, раскручиваются (`twist`, `spin`) и раскрываются в 4-рукавную галактику.
   - Деликатность: округлые рукава + постоянное вращение + растворение в вихрь = солнечное колесо/галактика, НЕ статичная эмблема.
- Стадии — окна `smoothstep` по `uh`; нарисованы вне множителя `pres`, чтобы ранние стадии были видны сразу при подходе.
- **Цветок Жизни — ВЕРНУЛИ** (2026-06-08): деликатная золотая мандала из 19 пересекающихся кругов на гексарешётке + внешнее кольцо, вплетена ЗА космогонией; проявляется по приближению (`smoothstep(0.14, 0.38, uh)`), масштаб `u = 0.165`, яркость ~0.42, аддитивно. Helper `folCirc` починен на возрастающий порядок краёв `smoothstep` (было `smoothstep(w, 0.0, …)` — на части GPU линии гасли, вероятно поэтому он и пропадал).

### Звук и эффекты входа
- Ambient-гул + чайм (Web Audio), кнопка mute ♪ (слева сверху). Звук стартует только по жесту.
- Влёт в сферу при клике (камера ныряет, CSS-warp).
- Рождение искры на старте (вуаль + вспышка).
- Хвост-комета за курсором (6 сегментов).
- Редкая падающая звезда (раз в ~18–40 сек).
- Всё уважает `reduced-motion`.

### i18n / UI
- Переключатель RU / EN (золотой), подпись «Войди в сферу».

### Мобильный вход (touch) — 2026-06-08
- **Одиночный тап → рождение «руки-свободны»**: палец можно убрать, космогония разворачивается сама за ~3.2 с (точка → круг → пульсация → 1-я линия → 2-я → 9-угольник → коловрат/галактика) и ЗАМИРАЕТ полностью рождённой; **вход — по второму тапу** (в любой момент).
- **Удержание → рождение под пальцем** (~1.6 с): додержал до конца → вход; отпустил рано → сворачивается; отпустил за порогом `0.45` → вход.
- Стадийная хаптика (Android): тики на `[0.16, 0.37, 0.57, 0.80]`.
- Двухстадийная подсказка (только на тач): «Коснись сферы» → «Коснись ещё раз, чтобы войти» (`touchHint` / `tapAgainHint` в `core.ts`).
- `onClick` не срабатывает на touch (`lastPointerType`) — нет случайного двойного перехода; вход на телефоне строго через тап-логику.
- Перф на телефонах: `dpr [1, 1.75]`, MSAA выкл (`scene.tsx`); 3D больше не глушится на 4-ядерных — порог fallback ослаблен до `cores <= 2 || mem <= 1` (`core.ts`).
- Гироскоп-параллакс (iOS-разрешение по первому касанию) + хаптика входа `[0, 18, 55, 30]`.

## 🗺️ Карта дальнейших изменений (roadmap)

1. **Узел 2 — Яйцо-лобби**: 4 портала-окна (вправо→3 Хроника, вниз→4 Daimon, влево→5 Душа, вверх→6 Настолка Ра); центр растёт со Светом; угловые кнопки; сверить с Васту.

3. **Тюнинг звезды/космогонии**: скорость вращения галактики, моменты появления стадий (сейчас крест ~60%, вихрь ~75% подхода), закрученность рукавов, толщина линий.
4. **Вернуть Bloom/EffectComposer** когда перф стабилен (postprocessing 3.0.4 ↔ three 0.184 ↔ fiber 9.6).
5. **Чистка**: мёртвый `istok.tsx`, неиспользуемые `artReady`/`nebulaBgStyle`/`.nebulaBg`, `bilingualCaption`.
6. **Позже**: LLM+аудио VoiceOfTruth, iOS gyro, swisseph, прогрессия Daimon, узлы 6–7 (Световая настолка, Держава Ра), миграция legacy→модули, интеграция в index.html.

## 🔧 Ручки тюнинга (быстрая справка)

- **Острота/форма звезды**: количество и шаг лучей в блоке (b) `RING_FRAG`.
- **Размер ободка**: `R` (сейчас 0.84); **прозрачность центра**: `smoothstep(0.34, 0.60, r)`.
- **Толщина линий**: `lw` (0.014 + 0.006·uh).
- **Моменты стадий космогонии**: окна `gDisk/gPoint/gHoriz/gVert/gWheel` (smoothstep по `uh`).
- **Вращение/закрутка галактики**: `spin` и `twist` в стадии (5).
- **Радиус внутренней сцены**: `ir` (0.50).
- **Цветок Жизни**: масштаб `u` (0.165), окно появления `smoothstep(0.14, 0.38, uh)`, яркость 0.42 — блок FLOWER OF LIFE в `RING_FRAG`.
- **Мобильный вход** (`EntryScreen.tsx`): скорость авто-рождения `0.0052` (~3.2 с), порог «быстрого тапа» `300 мс`, порог входа по удержанию `0.45`, моменты хаптики `STAGE_TICKS = [0.16, 0.37, 0.57, 0.80]`; перф — `dpr`/`antialias` в `scene.tsx`, порог fallback в `detectLowPower` (`core.ts`).
