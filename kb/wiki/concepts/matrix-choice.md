# Выбор Матрицы

Создано: 2026-05-26

## Принцип

Свобода выбора (не навязывание). Игрок сам выбирает матрицы.

## Матрица = Культура

Матрица — это культура, через призму которой игрок взаимодействует с игрой. У нас ведическое ядро (21 агент), матрицы надеты на ядро. Игрок изначально вшит в ведическую историю, но не знает этого (майя).

## Механика выбора

### Старт
- Игрок выбирает 3 основные матрицы из 33
- Только 3 — чтобы не засорять экран
- Эти 3 определяют начальный путь

### Прогресс
- Дошёл до определённого уровня в 3 матрицах → открываются ещё 2
- Упоминание матрицы в пространстве души → возможность открыть

### Смена
- Можно поменять матрицы, если не понравилось
- Не навязываем путь

## Путь через матрицу

Выбор матрицы → прохождение культуры → ключи → знания → образование → карточки → пассивный свет от энергии, с которой взаимодействовал, почувствовал, узнал.
