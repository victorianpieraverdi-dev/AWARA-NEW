# Лобби 3D — портал ИГРЫ и интерактив разделов

> Экспериментальный 3D-вариант лобби. **Отдельный файл** `lobby-3d.html`.
> `index.html` и `lobby.html` НЕ трогаем — это эталон/живая игра.
> `entry.html` «войти →» по-прежнему ведёт на `index.html` (не меняем до полного паритета).

## Что это
3D-яйцо Хираньягарбха на Three.js (r0.157) в центре лобби, RA-ядро в центре.
Медленное авто-вращение + drag. Тап по яйцу → выезжает единая панель из 4 разделов
(ИГРА / ХРОНИКА / ОБМЕН / ОБЩЕНИЕ). Обвязка (космос, шапка, статусбар, язык RU/EN,
паспорт, меню) — идентична `lobby.html`.

## Главный принцип дизайна
**«Чем больше света — тем круче интерфейс».**
Эффекты масштабируются от `totalLight` игрока (читается из `localStorage.awara_v255_state`):
```
lightF = clamp( log10(1+totalLight) / 4 , 0..1 )
```
Чем больше света → глубже наезд камеры, ярче ядро, плотнее волна, шире раскрытие портала.
Идея: лобби буквально расцветает по мере прокачки.

## Рождение сферы — первый контакт (вход) ✅

**Принцип:** при загрузке лобби сфера не «висит», а **рождается из пустоты** за ~4.5с (мистическое уплотнение из тумана). Решает проблему статичного белого обруча на старте — теперь вся структура проявляется из тьмы.

- `birthMat` — ShaderMaterial (additive, FrontSide) на отдельной сфере `birthShell = eggGeo(1.0)` в `eggGroup`.
  - **Мягкий Fresnel** края: `pow(1-abs(dot(vN,z)), mix(2.2,4.0,u_birth))` — края смягчены в начале, обостряются к финалу.
  - **Глубокий 3D-шум (fBm, 4 октавы)** `fbm(vP*1.8 + время)` — «дыхание спящего яйца», туман гуще в начале (`fog *= 1 - u_birth*0.55`).
  - **Бинду**: `smoothstep(0.34,0,length(vP.xy)) * u_birth` — свет стягивается в центральную точку к финалу.
  - `u_birth` 0→1: при 0 сфера полностью растворена (i=0), к 1 — плотная сфера с острым краем и ярким Бинду.
- **Драйвер:** таймер `life` в animate (`birth = smoothstep(life/4.5)`) — работает без зависимостей.
  - GSAP-хук готов (`gsap.fromTo(birthMat.uniforms.u_birth, …, {duration:4.6, ease:'power2.inOut'})`), активируется автоматически, если подключить GSAP. ⚠️ Сейчас GSAP в проекте НЕТ — драйвером служит таймер life (fallback обязателен, иначе сфера осталась бы невидимой).
- **Рождение структуры:** скорлупа (`shell`), решётка (`merMat`), ореол (`raHalo`) и все кольца (`raRing1/2`, `ring3a/b`, `ring2/2b`) умножаются на `birth` → проявляются вместе со сферой, без статичного обруча.
- `bloom = max(0, 1 - birth*1.2)` — золотая вспышка-рождение, гаснет к ~3.7с (усиливает ядро RA, ореол, пыль).

## Интерактив: каждый раздел отражается в сфере
Состояние FX: `window.__awaraFx = { hover, base }`; значения `cur.{game,chronicle,exchange,social}`
плавно (lerp 0.05) тянутся к цели. Наведение на кнопку раздела (`mouseenter`) → hover-эффект,
клик → закрепляет (toggle) + `awara:section`.

### ИГРА → ПОРТАЛ (текущая фишка)
При наведении в центре яйца **раскрывается светящаяся воронка-туннель** — ощущение входа в портал:
- `portalMat` — ShaderMaterial (additive, depthTest:false), плоскость 2.2×2.2 в центре сцены.
  - Полярные координаты: `r`, угол `a`; вихрь `sin(r*16 - uTime*4 + a*2)`.
  - `uOpen` (= gEase) раскрывает апертуру: яркое кольцо `edge` на радиусе раскрытия +
    светящаяся горловина `throat` внутри (голубой → бело-сияющий центр).
  - `portal.scale = 0.25 + gEase*(1.05 + lightF*0.35)`, медленное вращение `rotation.z = at*0.12`.
- **Наезд камеры** «в портал»: `camTarget = 4.3 - gEase*(0.4 + lightF*0.45)`.
- **Вдох**: `eggGroup.scale = 1 - gEase*0.04`, скорлупа раскрывается `shell.opacity = 0.3 - gEase*0.2`.
- **Ядро RA** разгорается как сердце портала (`uBoost = gEase*(0.5 + lightF*0.7)`).
- **Тонкая волна** по сфере (`waveMat`, полоса 0.03 + светлое ядрышко 0.012).
- **Виньетка** `.dive-veil` затемняет края — фокус на воронку (`stage.diving` при gIn>0.1).
- `gEase = smoothstep(cur.game)` — мягкий заход/выход без рывков.

### ХРОНИКА → аура + спутники
Ореол `auraMat` (BackSide, fresnel) вокруг яйца + 5 спутников `sats` на орбитах,
собирающих информацию (лучи `satBeams` спутник→центр). Дышит `1+sin(at*0.6)*0.018`.

### ОБМЕН / ОБЩЕНИЕ → нейронная сеть
Симметричная сеть по **икосаэдру** (12 вершин φ-пропорции + центр), `EGG_FILL=(0.8,1.08,0.6)` —
занимает больше пространства внутри яйца. ОБМЕН — тонкие связи + бегущие искры `pulses`;
ОБЩЕНИЕ — мягко светятся узлы-нейроны.

## История правок (этой сессии)
1. Базовый «вход в сферу» при наведении на ИГРУ (наезд камеры + вдох + ядро + виньетка), масштаб от света.
2. Сделал тоньше/изящнее (smoothstep, тонкая волна, мягкая виньетка) — оказалось СЛИШКОМ незаметно.
3. **Портал** — раскрывающаяся воронка-туннель + усиленный наезд + затемнение краёв. ✅ принято («супер»).
4. Убран статичный белый обруч на наведении (портал стал click-only) + добавлено **магическое свечение всей сферы** (`magicMat`) и **золотая пыль** (140 motes). ✅
5. **РОЖДЕНИЕ сферы при входе** (`birthMat`: Fresnel + 3D-fBm + `u_birth` 0→1 за ~4.5с, бинду в центре, структура проявляется из тьмы). ✅ принято («очень даже ничего»).
