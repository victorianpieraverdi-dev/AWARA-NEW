# Космический дизайн модульного лобби (lobby.html) — 2026-06-02

> Статус: черновик/полигон. `index.html` — эталон и живая игра, НЕ трогаем.

## Зачем переделали
Предыдущий lobby.html был «плоским» и не похож на старый сайт. Пользователь попросил «переделать под наш крутой космический».

## Почему не скопировали 1:1 из index.html
- index.html = 1.16 МБ / 16000+ строк; MCP-инструмент БЕЗ grep и без чтения по диапазону строк (только head/tail). Чтение head упало (fetch failed).
- Настоящий RA в index.html рисуется на `<canvas id="hiranya-canvas">` встроенным JS — код недоступен для точного копирования.
- Модульные CSS (`typography.css`, `main.css`, `components.css`) БИТЫЕ: ссылаются на несуществующие переменные (`--font-body`, `--awara-gold-primary`, `--space-md`, `--glow-gold-bright`, `--awara-bg-dark`). Единственный цельный файл — `variables.css`.

## Что сделано (новый lobby.html)
Собрана космическая мандала с нуля, на точной палитре из variables.css:
- **Фон `#cosmos`** (full-screen canvas): 160 мерцающих звёзд + 5 пульсирующих туманностей (пурпур/синь/золото).
- **Янтра `#hiranya`** (canvas): радиальное золотое свечение, 72 луча (вращаются), 12 лотосных лепестков (контр-вращение), 4 кольца, 48 точек на ободе, пульсация.
- **Солнце RA** в центре (`.central-sun [data-central-sun]`): дышит (`@keyframes breath`), золотой bloom.
- **4 врата по углам** (`.gate[data-section]`): ИГРА(синий ◉), ХРОНИКА(золото △), ОБМЕН(бирюза ⚜), ОБЩЕНИЕ(пурпур ✥). Свечение на hover, RU/EN data-attrs.
- **Шапка**: badge, AWARA (Cinzel Decorative градиент), ruler «sapta yoga», счётчик света.
- **Статусбар**: light · ранг · агент + RU/EN переключатель (`awara_lang`).
- **Кнопки**: ☰ Меню (showModal+getMenuState), Паспорт души, ← Исток (entry.html).
- **Мобильный фоллбэк** @media≤600px: врата в столбик, мандала меньше.

## Логика (не изменилась)
Модули: lobby-module(initLobby,showPassport), game-module(initGame,showGameMenu), menu-module(getMenuState), ui-module(showModal,showToast). Состояние — общий ключ `awara_v255_state`. Событие `awara:section` → game ? showGameMenu() : showToast(«скоро»).
