# Исток — вход Души

> Возвращение к самому началу. Вход для неразвитого игрока, который ещё не знает правил космической игры, но делает выборы исходя из своего опыта. Разговор на равных — не свысока.

Концепция выведена из анализа всей пройденной игры: первый экран — это «начало», свёрнутое из целого. Все остальные окна игры разворачиваются уже из него. Вход — это не экран со сферами, а **первая запись в живой журнал**.

---

## Сценарий-медитация входа (v2 — АКТУАЛЬНЫЙ)

Тьма → дыхание → касание → первая искра света → покой. Без диалога и форм. Тон: тихо, медитативно.

**0. Тьма (0–2.6 сек)**
- Почти чёрный дышащий космос (#02010a), мягкое мерцание звёзд.
- В центре еле тлеет «семя света» (золотая точка), дышит.
- Ничего не просят. Просто тишина.

**1. Дыхание**
- Проявляется строка (курсив): «Здесь темно. Вдохни — и коснись тьмы.»
- В центре — ведущее кольцо, расширяется/сжимается (~7 сек, вдох/выдох), помогает дышать.
- Над кольцом тихое слово «В Д О Х» / «В Ы Д О Х» в ритме дыхания.

**2. Касание → искра**
- Касание в любой точке рождает искру света там, где коснулся (искра из пальца).
- Искра расцветает и плавно оседает в центр (~2.6 сек), тьма теплеет.

**3. Покой**
- В центре остаётся дышащий уголёк.
- Проявляется «Свет начался.»
- Ненавязчивое «войти →» ведёт в лобби (index.html).

Жест бессловесный (дыхание + касание), текста минимум. Игра дополняет, не инструктирует.

---

## Решения (опрос 31.05)

- Первое действие игрока = **вдох + касание (искра из пальца)**. Не текст, не форма, не диалог.
- Сферы / Даймон / полярность **не видны на входе**; Даймон и сферы проявляются позже, когда игрок формирует их изнутри.
- **Ида/Пингала (Луна/Солнце) убраны со входа** — игрок поймёт их выше, на уровнях.

## Что убрано со входа (было в v1 / T-631)

- Вертикальный столбик сфер Разум·Эго·Сердце·Воля.
- Слот Даймона с «+» и тост «Создай Даймона».
- HUD Луна/Солнце, краевые ярлыки Ида·Нисходящее / Пингала·Восходящее.
- Срединная ось Сушумна.

Это не выброшено из игры — перенесено «на потом» (см. раздел «Отложено»).

## Отложено / для глубоких уровней (сохранено, НЕ на входе)

### Ось сфер (исходный порядок сверху вниз)
1. **Разум** (зелёный) — вверху
2. **Эго / Личность** (фиолетовый)
3. **Сердце** (роза) — над Волей
4. **Воля** (голубой)
5. **Даймон** — слот снизу

### Полярность (полная модель)
- Левая = Луна / Ида (нисходящее, женское, тьма); правая = Солнце / Пингала (восходящее, мужское, свет).
- Дуальности (логическое ↔ фантазийное и пр.) ложатся на ту же ось.
- Со временем полюса объединяются → центральный канал **Сушумна**.
- Заполнение левая/правая ИЛИ единый текст → ИИ распределяет по сферам (LLM отложено, сначала эвристика).

### Душа v2 — порядок развития
1. Два полюса → 2. объединение полюсов → 3. привязка к уровням (каждый уровень = чакра-мир) → 4. ИИ-слой.

### Даймон (полная модель)
- Обязателен, растёт вместе с игроком, связан с натальной картой; реагирует и вступает в диалог при заполнении сфер.

### Мифология (origin story)
- Предыстория тьмы→света: камни бесконечности / первичные камни на заре творения. Оформить в [[../mechanics/grayscale-to-color]] / философию проекта.

### Шахматная тема (позже)
- Игрок поляризуется, проходя чёрно-белые клетки выбора, выращивает энергетическую фигуру. См. [[black-and-white]].
