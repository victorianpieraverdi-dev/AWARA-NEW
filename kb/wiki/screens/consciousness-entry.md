# Экран входа "Сознание" (Consciousness Entry)

Создано: 2026-05-31
Статус: Концепция → Реализация
Приоритет: **0.1** (самый первый экран игры)

## Назначение

Самый первый экран AWARA — **до** инициации, **до** Души, **до** лобби.
Для подростка 13-15 лет, который ещё не знает правил космической игры.

## Философия

**"Вход Тьма — Выход Свет"**

Человек входит в игру "зелёненьким" — не развитым, не знающим, как устроена жизнь.
Игра обращается к нему как к другу, проводнику, который уже прошёл этот путь.

## Визуальная структура (сверху вниз)

```
┌─────────────────────────┐
│   РАЗУМ (зелёная)       │  ← Ум, интеллект, левое полушарие
├─────────────────────────┤
│   ЭГО / ЛИЧНОСТЬ        │  ← Кто я?
├─────────────────────────┤
│   ВОЛЯ (голубая)        │  ← Посередине между Разумом и Сердцем
├─────────────────────────┤
│   ДАЙМОН                │  ← Появляется после ввода данных
├─────────────────────────┤
│   СЕРДЦЕ (розовая?)     │  ← Душа, эмоции, правое полушарие
└─────────────────────────┘
```

## Механика

### 1. Приветствие
Дружеское обращение от ИИ-проводника (Искра? Свет Ра?):
- "Привет! Ты готов начать путь?"
- "Я твой проводник в этой игре. Давай познакомимся."

### 2. Ввод данных
Человек вводит информацию о себе:
- **Вариант A:** Свободный текст — "Расскажи о себе"
- **Вариант B:** Дата рождения + имя
- **Вариант C:** Серия вопросов (кто ты? что любишь? о чём мечтаешь?)

### 3. Распределение ИИ
ИИ читает текст и распределяет по сферам:
- **Левая часть (Разум/Ум):** логика, факты, анализ
- **Правая часть (Сердце/Душа):** эмоции, мечты, чувства
- **Воля:** намерения, цели, решения

Сферы начинают заполняться светом.

### 4. Появление Даймона
На основе введённых данных создаётся Даймон:
- Если есть дата рождения → натальная карта → форма Даймона
- Если нет → быстрый выбор или рандом
- Даймон появляется визуально, реагирует на заполнение сфер

### 5. Общение
Даймон и ИИ-проводник комментируют:
- "Вижу, у тебя сильное сердце!"
- "Твой разум ищет ответы..."
- "Воля пробуждается..."

### 6. Переход
После заполнения базовых сфер → переход в текущую инициацию (index.html)

2. **Ввод данных:** Вопросно-ответная игровая форма → смещение фокуса в натальную карту по желанию

   - Уровень всех сфер растёт
4. **Визуал сфер:** Начать с 2D → по мере роста света растут в 3D
5. **Мифология:** В виде игры, завлечения намёками, загадками
6. **Статус:** Это **новый первый уровень** и его основа
   - Другие экраны закрыты, пока этот не пройдётся

## Детальная механика (на основе ответов)

### Поток игрока

**Шаг 1: Приветствие**
```
Безличный мистический голос:
"Ты стоишь на пороге. Я — тот, кто уже прошёл этот путь.
Готов ли ты начать игру?"
```

**Шаг 2: Вопросно-ответная игра**
Серия магических вопросов (не просто анкета):
- "Веришь ли ты в судьбу?"
  - Да → акцент на натальную карту
  - Нет → акцент на свободу выбора
- "Можешь ли ты влиять на свой путь?"
- "Что сильнее в тебе — разум или сердце?"
- "Что ты ищешь в этой игре?"

Каждый ответ → сферы начинают заполняться светом.

**Шаг 3: Смещение в натальную карту (опционально)**
Если игрок верит в судьбу:
- "Хочешь узнать, что звёзды говорят о тебе?"
- Ввод даты/времени/места рождения
- Расчёт натальной карты → определение формы Даймона

**Шаг 4: Заполнение сфер**
ИИ распределяет ответы по сферам:
- Разум (зелёная): логические ответы, анализ
- Сердце (розовая): эмоциональные ответы, мечты
- Воля (голубая): намерения, решения
- Эго: самоопределение

Визуал: сферы растут от 2D плоских → 3D объёмные
Цвет: от ЧБ/серого → яркие цвета по мере света

**Шаг 5: Появление Даймона**
На основе:
- Натальной карты (если есть)
- Ответов на вопросы
- Баланса сфер

Даймон материализуется, приветствует игрока.

**Шаг 6: Мифологические намёки**
Через загадки и метафоры:
- "В начале была Тьма и Свет..."
- "64 клетки мироздания ждут твоих ходов..."
- "Каждый выбор — это шаг от Тьмы к Свету..."

Не прямой текст, а интерактивные намёки.

**Шаг 7: Переход**
После заполнения базовых сфер:
- "Первый шаг сделан. Теперь — инициация."
- Переход в index.html (текущая инициация)

### Визуальная прогрессия сфер

```
Уровень 0 (пустая):  ⚫ 2D плоская, чёрная
Уровень 1-25:        ⚪ 2D, серая → светлеет
Уровень 26-50:       🔵 начинает расти в 3D, цвет проявляется
Уровень 51-75:       🟢 3D сфера, яркий цвет
Уровень 76-100:      ✨ 3D сфера, светится, пульсирует
```
