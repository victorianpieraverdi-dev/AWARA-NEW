# Магические вопросы для экрана "Сознание"

Создано: 2026-05-31
Цель: Вовлечь подростка 13-15 лет в духовную игру через простые, но глубокие вопросы

## Принципы

1. **Не проповедовать** — спрашивать
2. **Не учить** — пробуждать
3. **Не пугать эзотерикой** — говорить на языке жизни
4. **Магия в простоте** — каждый вопрос = зеркало

## Вопросы (15 штук)

### Блок 1: Пробуждение (кто ты?)

**Q1: Приветствие**
```
Ты стоишь на пороге.
Я — тот, кто уже прошёл этот путь.

Готов ли ты начать?
[Да, готов] [Не уверен] [Что это за игра?]
```

**Q2: Судьба или выбор?**
```
Веришь ли ты, что твоя жизнь уже написана звёздами?
Или ты сам пишешь свою историю?

[Судьба решает] [Я выбираю сам] [И то, и то]
```
→ Да/И то = путь натальной карты
→ Нет = путь свободы

**Q3: Разум или Сердце?**
```
Когда ты принимаешь важное решение —
что говорит громче?

[Голова (логика)] [Сердце (чувства)] [Оба спорят]
```
→ Заполняет Разум или Сердце

**Q4: Свет или Тьма?**
```
Сейчас в твоей жизни больше света или тьмы?

[Много света] [Баланс] [Больше тьмы] [Не знаю]
```
→ Начальный уровень света души

### Блок 2: Поиск (что ищешь?)

**Q5: Зачем ты здесь?**
```
Что привело тебя в эту игру?

[Любопытство] [Ищу ответы] [Хочу измениться] [Просто так]
```
→ Заполняет Волю

**Q6: Твоя сила**
```
В чём твоя настоящая сила?

[Ум и знания] [Доброта] [Воля и упорство] [Творчество] [Не знаю пока]
```
→ Определяет архетип

**Q7: Твой страх**
```
Чего ты боишься больше всего?

[Одиночества] [Неудачи] [Потерять себя] [Боли] [Ничего не боюсь]
```
→ Тень, с которой работать

### Блок 3: Связь (кто с тобой?)

**Q8: Друзья**
```
У тебя много друзей или несколько близких?

[Много знакомых] [Пара близких] [Один лучший] [Пока один]
```
→ Социальная сфера

**Q9: Природа**
```
Когда ты на природе, что чувствуешь?

[Покой] [Силу] [Связь с чем-то большим] [Скуку] [Не бываю]
```
→ Связь с элементами

**Q10: Животные**
```
Если бы ты мог стать животным на один день — кем бы стал?

[Хищник (волк, тигр)] [Птица (орёл, ворон)] [Водное (дельфин, кит)] 
[Мудрое (сова, слон)] [Мистическое (дракон, феникс)]
```
→ Подсказка для формы Даймона

### Блок 4: Путь (куда идёшь?)

**Q11: Мечта**
```
Если бы ты мог изменить что-то в мире — что бы это было?

[Больше добра] [Больше справедливости] [Больше свободы] 
[Больше красоты] [Больше правды]
```
→ Высшая цель

**Q12: Время**
```
Где тебе комфортнее?

[В прошлом (воспоминания)] [В настоящем (здесь и сейчас)] 
[В будущем (мечты)] [Везде одинаково]
```
→ Временной фокус

**Q13: Тайна**
```
Веришь ли ты, что в мире есть магия?

[Да, точно есть] [Хочу верить] [Не знаю] [Нет, это сказки]
```
→ Открытость к мистическому

### Блок 5: Инициация (последний шаг)

**Q14: Имя**
```
Как тебя зовут в этом мире?

[Ввод имени или ник]
```
→ Персонализация

**Q15: Звёзды (опционально)**
```
Хочешь узнать, что звёзды говорят о тебе?

[Да, интересно] [Нет, я сам выбираю]
```
→ Если Да = натальная карта (дата/время/место рождения)
→ Если Нет = быстрый выбор Даймона

## Логика распределения

### По сферам:
- **Разум (зелёная):** Q3 (голова), Q6 (ум), Q13 (скептицизм)
- **Сердце (розовая):** Q3 (сердце), Q6 (доброта), Q9 (связь), Q11 (мечта)
- **Воля (голубая):** Q5 (зачем), Q6 (упорство), Q12 (фокус)
- **Эго (жёлтая?):** Q1 (готовность), Q7 (страх), Q8 (социум), Q14 (имя)

### По свету:
- Q4 прямо задаёт начальный уровень
- Позитивные ответы +10-20 света
- Негативные/неопределённые +5 света
- "Не знаю" = нейтрально, но честно → +10 света за честность

### По Даймону:
- Q10 (животное) = прямая подсказка царства
- Q6 (сила) = архетип (Guardian, Wanderer, Sovereign...)
- Q9 (природа) = элемент (Earth, Water, Fire, Air, Ether)
- Q15 (звёзды) = если Да → натальная карта → точная форма

## Тон голоса

**Безличный мистический проводник:**
- Не "я научу тебя", а "я прошёл этот путь"
- Не "правильный ответ", а "твой ответ"
- Не "ты должен", а "ты можешь"
- Не "эзотерика", а "жизнь"

**Примеры фраз между вопросами:**
- "Интересно... Твоё сердце говорит громче, чем ты думаешь."
- "Вижу свет в твоих словах."
- "Ты ищешь то, что уже внутри тебя."
- "Каждый ответ — это шаг из тьмы."

## Визуальное сопровождение

Пока игрок отвечает:
- Сферы медленно заполняются светом
- Фон меняется от тёмного к светлому
- Появляются звёзды, частицы света
- Музыка нарастает (от тихой к яркой)

## Следующий шаг

После Q15:
- Если выбрал натальную карту → ввод данных → расчёт → появление Даймона
- Если отказался → быстрый выбор из 3-5 форм Даймона по ответам
- Даймон материализуется, говорит первую фразу
- Переход: "Первый шаг сделан. Теперь — инициация."
