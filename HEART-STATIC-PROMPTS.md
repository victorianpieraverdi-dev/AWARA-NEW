# AWARA · Сердце Истока — промты для СТАТИЧНЫХ картинок

Цель: сначала сгенерить **статичный образ Сердца** (центральный символ всей игры). Оживление/анимация — позже. Механика плавки пока остаётся как есть.

---

## Общие правила (для всех вариантов)

- **Формат:** квадрат 1:1 (под орб). Для экрана Истока можно и 9:16.
- **Композиция:** строго по центру, симметрия, ядро в середине, вокруг — воздух/тьма (чтобы красиво вставало в круг).
- **Фон:** глубокий тёмный космос / чёрный, без текста, рамок, UI, подписей.
- **Стиль:** алхимия + космос, священная геометрия, объёмный свет, золото + фиолет (бренд Тигля).
- **Бренд-палитра:** золото #c9a84c / #ffd27a, фиолет #7b62c9 / #9d86e0, тёмный фон #0b0a2a → #04030f.
- **Генераторы:** Midjourney v6 / Flux / SDXL / DALL·E 3. Промты на английском, концепт по-русски.
- **MJ-хвост (если Midjourney):** добавляй ` --ar 1:1 --style raw --v 6` (или `--ar 9:16`).
- **Куда класть:** `C:\AWARA\exports\generated_cards\heart\heart_A.webp …` (имена по варианту).

## Как выбирать
- A/B — «космическое ядро» (мощно, универсально).
- C/F — мягче/светлее, ближе к канону «рай на земле».
- D/E — явный образ «сердца/тигля» (ближе к теме плавки).

Совет: сгенерируй по 1–2 картинки на каждый вариант, отбери лучший — он станет базовым образом и для экрана Истока, и (позже) базой для анимации.
