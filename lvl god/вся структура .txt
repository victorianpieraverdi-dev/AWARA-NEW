# 🕉️ СУТЬ И ФИЛОСОФИЯ

## Главная Идея

**AWARA** — это живая система духовной трансформации, симулятор эволюции сознания от Кали-юги (хаос, невежество, материализм) к Сатья-юге (свет, просветление, единство).

### Цель Игры

**НЕ зациклить игрока**, а:
- Дать принципиальную базу божественных принципов
- Сделать технические устройства друзьями и помощниками
- Освободить от любого типа рабства и невежества
- Помочь стать Буддой-царём, Богом, Аватаром Единого или просто хорошим человеком

### Путь Игрока

```
КАЛИ-ЮГА (тьма) → НАПОЛНЕНИЕ СФЕР → САТЬЯ-ЮГА (свет)
     ↓                    ↓                    ↓
  Невежество         Осознание           Просветление
  Статика            Движение            Жизнь
  Потребление        Обмен               Творение
```

# 🎮 ИГРОВАЯ СТРУКТУРА

## Архитектура Сфер

### 12 Основных Сфер (Мерностей)
Расположены на 9 спектральных орбитах (от красной до золотой):
- **Назначение**: Путь эволюции сознания игрока
- **Поведение**: Вращаются вокруг центра РА
- **Эволюция**: Растут от статики к живым сущностям

### 3 Тримурти (Божественные Силы)
- **Шива** — Разрушение (красный нимб)
- **Вишну** — Сохранение (синий нимб)  
- **Брахма** — Творение (золотой нимб)
- **Поведение**: Вращаются в 3D вокруг РА, создавая объёмную сферу
- **Расположение**: Ближе к центру, более трансцендентны

### 1 Источник РА (Центр)
- **Форма**: Золотой снитч с крыльями
- **Функция**: Центральное божественное начало
- **Поведение**: Пульсирует, излучает волны света

### 4-8 Личных Сфер
- **Назначение**: Личные цели, особые аспекты бытия игрока
- **Появление**: По запросу или достижению

### Всплывающие Культурные Сферы
- **Системы**: Буддизм, Даосизм, Генные Ключи, Джойтиш, йога...
- **Эффект**: Добавляют цвета и краски культуры в мир игрока
- **Механика**: Углубление в традицию меняет визуал и звук

### Дополнительные Элементы
- **Сингулярность** (чёрная дыра) — портал в системы метрик
- **Судьба** — личная сфера высокого уровня
- **Искра** — маленький свет, гуляющий по экрану, описывает состояние сфер

### Астрологические Влияния
- **Планеты** — каждый день новые влияния
- **Ветры судьбы** — помогают или препятствуют

## Три Стадии Эволюции

### I. ДРЕВО (Уровни 1-3) — Человек и его мир
**Визуал:**
- Мёртвые, статичные сферы
- Малый огонь/вода/воздух/земля
- Геометрия простая

**Концепция:**
- Тело как сервер восприятия (hardware)
- Осознание физического уровня
- "Ламповый экран" → "HD экран"

**Механика:**
- Игрок начинает наполнять сферы
- Сферы тускнеют при игнорировании
- Звук тихий, примитивный

### II. КРИСТАЛЛ (Уровни 4-6) — Природа и служение
**Визуал:**
- Сферы оживают, начинают двигаться
- Стихии растут в живые планеты-сущности
- Геометрия усложняется
- Формы: ромб, звезда

**Концепция:**
- Системы знаний как драйверы (software)
- Служение людям
- Отношения с природой

**Механика:**
- Сферы реагируют на прикосновения
- Появляются шлейфы за сферами
- Музыка переходит к природным звукам (птицы)

### III. КОСМОС (Уровни 7-9) — Дух и Вселенная
**Визуал:**
- Сферы = отдельные живые космические сущности
- Максимальная динамика и сложность
- Шлейфы рисуют Цветок Жизни
- Форма: кристалл (гексагон)

**Концепция:**
- Космическое сознание
- Аватар Единого
- Высшее Я

**Механика:**
- Полная интерактивность
- Космическая музыка
- Создание света (игрок больше создаёт, чем потребляет)

# ⚙️ МЕХАНИКИ И СИСТЕМЫ

## Система Уровней

### 9 Уровней = 7 Чакр
```
Уровень 1 (Красный)    → Муладхара   → 256 Hz
Уровень 2 (Оранжевый)  → Свадхистана → 288 Hz
Уровень 3 (Жёлтый)     → Манипура    → 320 Hz
Уровень 4 (Зелёный)    → Анахата     → 341 Hz
Уровень 5 (Голубой)    → Вишудха     → 384 Hz
Уровень 6 (Синий)      → Аджна       → 426 Hz
Уровень 7 (Фиолетовый) → Сахасрара   → 480 Hz
Уровень 8 (Белый)      → Выше чакр   → 512 Hz
Уровень 9 (Золотой)    → Единство    → 528 Hz (частота любви)
```

### 4 Формы (Круги)
1. **Круг** (Сфера) — базовая форма
2. **Ромб** — кристаллизация
3. **Звезда** — излучение
4. **Кристалл** (Гексагон) — космическая форма

**Переход между формами**: При повышении "круга" (К+ кнопка)

## 💎 Светкоин (Валюта Совести)

### Начисление
- ✍️ **За записи** в сферы (описания, осознания)
- 📅 **За регулярность** (дисциплина)
- 🎯 **За глубину** (искренность, проживание)
- 💫 **Милость игры** (бонусы за прогресс)

### Свойства
- **Нельзя подделать** — подтверждается "Смарт-контрактом Сердца"
- **Накапливается** в Банке Света
- **Используется** на маркетплейсе (услуги, творчество)
- **Генерируется** через NFT (по силе и редкости)

### Формула
```
Светкоин = Осознанность × Регулярность × Благотворительность
```

Чем выше уровень → тем больше игрок отдаёт всем → тем меньше потребляет

## 🎨 Система Стихий и Животных

### Стихии → Тотемы
По мере роста уровня, стихии превращаются в живых животных-архетипов:

**Огонь** 🔥
- Уровень 1-3: Маленький огонёк
- Уровень 4-6: Тигр
- Уровень 7-9: Лиса/Феникс

**Вода** 💧
- Уровень 1-3: Капля
- Уровень 4-6: Кит
- Уровень 7-9: Олень

**Воздух** 🌪️
- Уровень 1-3: Ветерок
- Уровень 4-6: Сова
- Уровень 7-9: Бабочка

**Земля** 🌍
- Уровень 1-3: Камень
- Уровень 4-6: Медведь
- Уровень 7-9: Слон

**Эфир** ✨
- Уровень 1-3: Свет
- Уровень 4-6: Волк
- Уровень 7-9: Феникс

### Гибридные Архетипы
Формируются из статистики игрока (например, Кит + Тигр = уникальный визуал)

## 🧠 ИИ-РА (Искусственный Интеллект)

### Функции
- 📊 **Даёт права на переход** между уровнями (когда сферы наполнены смыслом)
- 💬 **Советует** по игре и жизни
- 🔮 **Задаёт загадки** (проверка осознанности)
- ⚖️ **Оценивает искренность** записей
- 🌟 **Управляет милостью** игры

### Искра (Помощник ИИ)
- Белая светящаяся точка, летающая по экрану
- Заглядывает в сферы
- Описывает их состояние
- Даёт подсказки

### Критерии Перехода
```
Если игрок НЕ прогрессирует → сферы высыхают → мир тускнеет
Если игрок наполняет сферы стихиями → рост чувств → природа
Если игрок наполняет состояниями и стихиями → усложнение
```

### Тримурти

```javascript
// 3D вращение вокруг РА
const rotX = state.t * 0.5 + baseAngle;
const rotY = state.t * 0.3 + baseAngle * 0.5;
const x3d = Math.cos(rotX) * orbitRadius;
const z3d = Math.sin(rotY) * orbitRadius;
const depth = (z3d / orbitRadius + 1) / 2; // 0 to 1
size = 48 * (0.7 + depth * 0.6); // перспектива
opacity = 0.6 + depth * 0.4;     // глубина
```
