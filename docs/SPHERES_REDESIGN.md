# AWARA — РЕДИЗАЙН СФЕР И ОНБОРДИНГА

## Концепция

Игрок заходит → видит не сложную панель с кучей кнопок, а **одну сферу** — свою первую. Делает простое действие → открывается вторая сфера → третья → четвёртая. К концу он видит **живую нейросеть своих сфер** — они связаны, пульсируют, отражают его свет и путь.

---

## ПОШАГОВЫЙ ОНБОРДИНГ

### Шаг 0: Вход
```
«Ты вошёл в пространство души. Здесь живут твои сферы.
Начни с первой — Основа.»
→ Одна сфера видна, остальные затемнены.
```

### Шаг 1: ОСНОВА (первая сфера)
```
«Это твоя Основа. То, на чём ты стоишь.
Создай первую подсферу — напиши что ты планируешь на неделю.»
→ Игрок создаёт подсферу внутри «Основы»
→ Сфера активируется, светится
→ Открывается вторая сфера: СЕРДЦЕ
```

### Шаг 2: СЕРДЦЕ
```
«Основа активна. Теперь Сердце — что ты делаешь прямо сейчас?
Запиши одно действие сегодняшнего дня.»
→ Игрок создаёт подсферу в «Сердце»
→ Открывается третья сфера: РАЗУМ (ГОЛОВА)
```

### Шаг 3: РАЗУМ / ГОЛОВА
```
«Основа + Сердце активны. Теперь Разум — твоё видение.
Опиши образ будущего, к которому ты идёшь.»
→ Игрок создаёт подсферу в «Разуме»
→ Все 3 сферы активны
→ Появляется подсказка: «Свяжи сферы»
```

### Шаг 4: СВЯЗИ
```
«Твои сферы могут быть связаны. Соедини Основу с Сердцем:
как твои планы (Основа) влияют на действия (Сердце)?»
→ Игрок создаёт связь между сферами
→ Связь визуализируется как линия/поток
→ Открывается 4-я сфера: СОТРУДНИЧЕСТВО
```

### Шаг 5: СОТРУДНИЧЕСТВО
```
«Теперь ты готов к общей практике.
Пригласи другого игрока или создай общую сферу.»
→ Открывается social-слой
→ Все 4 сферы + связи образуют нейросеть
```

## ВИЗУАЛИЗАЦИЯ: КАРТА СОЗНАНИЯ (MIND MAP)

### Внешний вид
```
         ┌──────────┐
         │  РАЗУМ   │ ← 3-я сфера
         │ (голова) │
         └────┬─────┘
              │ (связь)
         ┌────┴─────┐
    ┌────┴────┐     │
    │ СЕРДЦЕ  │─────┤ ← 2-я сфера  
    └────┬────┘     │
         │ (связь)  │
    ┌────┴────┐     │
    │ ОСНОВА  │     │ ← 1-я сфера
    │ (ноги)  │     │
    └─────────┘     │
                    │
              ┌─────┴──────┐
              │ СОТРУДНИ-  │ ← 4-я сфера
              │ ЧЕСТВО     │
              └────────────┘
```

### Стиль
- **Тёмный фон** + золотые узлы (сферы)
- **Линии-связи** — светящиеся, разной толщины (чем сильнее связь, тем ярче)
- **Новые связи** создаются перетаскиванием (drag from sphere A to sphere B)
- **ЭМ-поле** — лёгкое золотое свечение вокруг всей структуры
- **Пульсация** активных сфер, тусклое свечение неактивных
- Похоже на: **нейросеть**, **mind map**, **созвездие**

## ПОДСКАЗКИ И ОБУЧЕНИЕ

### Контекстные подсказки (всплывают сами)
```
«Создай подсферу» — когда сфера пуста
«Свяжи с Основой» — когда есть 2+ активные сферы без связей
«Уникальная связь!» — когда игрок создал новую комбинацию
«Практика дня: укрепи связь Основа→Сердце» — daily quest
```

### Принцип
- Подсказка появляется ТОЛЬКО когда релевантна
- Не больше 1 подсказки за раз
- Исчезает после действия
- Формат: 1 строка, золотой текст, мягкая анимация

## ИНТЕГРАЦИЯ С ТИГЕЛЕМ

Тигель становится **частью сфер** — не отдельным экраном:

```
Сфера «Сердце» → внутри поле «Тигель»
  «Что ты сделал сегодня?» ← короткая запись
  → Авто-теггинг У-Син
  → Свет начисляется в сферу
```

ИЛИ:

Тигель остаётся отдельной вкладкой, но **результаты Тигеля отображаются в сферах**:
- Баланс Ида/Пингала → видно в центре карты сознания
- Свет за день → распределяется по активным сферам
- Стихия дня → подсвечивает соответствующую сферу
