# AWARA — Аудио-гайд и промты

Звук игры AWARA: сакрал-космос, путь по узлам 0 -> 7, чакральная восходящая лестница.
Используем гибрид: ИИ (Suno) генерирует атмосферу, живое (голос, чаши, field recording) даёт душу.

> Suno лучше понимает стиль-теги на английском, поэтому поле **Style** даётся на английском,
> а биджа-мантры / пояснения — на русском. Для чистого инструментала включай тумблер **Instrumental**.

---

## 0. Принцип натуральности (главное)

Формула: **сгенерируй атмосферу -> положи сверху один живой дышащий звук -> объедини общим ревером -> расшатай тайминг.**

- ИИ = основа (дрон, пэды, хор, космос). Живое = душа (голос, поющая чаша, гонг, дыхание, ветер/вода/огонь).
- Ухо прощает синтетику в фоне, если на переднем плане есть хоть один живой звук.
- В Style тяни к органике: `organic, acoustic, live recording feel, room ambience, breathy vocals, natural reverb, analog warmth, imperfect timing, humanized, unquantized, tape saturation`.
- Избегай «цифрового»: `EDM, heavy quantize, autotune, hyperpop, loud master`.
- Натуральность внутри Suno: `432Hz`, `no drums`, `slow swells, breathing dynamics, rubato`, `minimal, spacious, sparse`, `imperfect, human, intimate, close-mic`.

### Гибридный пайплайн
1. Suno -> базовый слой (несколько вариантов, берёшь лучший).
2. Stem split (выгрузка стемов / разделитель дорожек).
3. DAW (Cakewalk / Reaper / Audacity) -> раскладываешь стемы + свои живые записи.
4. Обработка: convolution reverb (реальный импульс зала/храма) на всё; tape/analog saturation; сдвиг живых элементов от сетки на +-10..30 мс.
5. Луп без стыков: режь по нулям волны / кроссфейд 1-2 с.

### Для AWARA
- ИИ: дрон + хор + космические пэды.
- Живое: твой голос с биджа-мантрами (ЛАМ/ВАМ/РАМ...), поющая чаша, дыхание.
- Голос автора нельзя сгенерировать одинаково — он делает игру личной.

## 5. Привязка к коду

- `core/ai/VoiceOfTruth.ts`: `CHAKRA_ELEMENT_MAP = [earth, water, fire, air, ether, ether, ether]`.
- `core/types.ts`: `PlayerState.chakras[7]`, `light` (старт 0).
- Идея: активная чакра игрока -> её тема; по росту Света кроссфейд снизу вверх (396 -> 963 Hz) = музыкальная «лестница пробуждения».
- Полный «Гимн Пути» -> финал, узел 7 (Держава Ра), как гимн завершения пути.
- Можно резать гимн на 7 стемов по частям и проигрывать активную чакру.

## 6. Чеклист настроек Suno

- Style влияние ~60-70% (атмосфера, а не «песня»).
- Для лупа: генерируй несколько, бери самый ровный кусок без резких переходов.
- `432Hz` / `no drums` = медитативность.
- Частоту указывай прямо в Style (`528Hz`) — это музыкальная окраска, не лабораторный тон (для точного тона нужен tone-generator).
- Генерируй 2-3 варианта каждой темы, выбирай тот, что лучше «дышит».

## 8. Аффирмации / интенции по узлам (настройка под зону работы)

Короткая настройка перед работой над конкретным узлом — входишь в состояние самой зоны.

- **Узел 0 — Искра / Вход** (earth-void)
  > «Я начинаю из тишины. Первая строка — это искра. Я зажигаю её спокойно.»
  Style-окраска: `dark void drone, single tender spark, reverent, minimal`

- **Узел 1 — Вселенная-тороид** (расширение)
  > «Я вижу целое. Структура дышит и вращается. Я держу архитектуру в уме без спешки.»
  Style: `expansive cosmic awe, slow rotation, vast, structural clarity`

- **Узел 2 — Яйцо-лобби** (выбор пути)
  > «Четыре двери открыты. Я выбираю одну и иду в неё целиком.»
  Style: `calm sanctuary, four directions, contemplative, grounded choice`

- **Узел 3 — Хроника Матрицы** (структура/данные)
  > «Я распутываю узор. Каждая связь встаёт на место. Ясность вместо хаоса.»
  Style: `intricate indigo patterns, focused analytical drone, crystalline, precise`

- **Узел 4 — Daimon** (огонь/трансформация)
  > «Я кую форму из сырого. Сопротивление — это материал. Я преобразую его.»

- **Узел 5 — Душа** (вода/чувство)
  > «Я слушаю, что хочет ожить. Я веду мягко, доверяю течению.»

- **Узел 6 — Световая настолка Ра** (воздух/игра/правила)
