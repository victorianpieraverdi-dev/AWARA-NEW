# AWARA — ДИЗАЙН ИГРЫ (GAME DESIGN DOCUMENT)

Видение Pavel'а от 21.05.2026. Структурировано по слоям и уровням.

---

## АРХИТЕКТУРА: 4 СЛОЯ ИГРОКА

```
                    ┌─────────────────────────┐
                    │  4. ИГРА МИРОЗДАНИЯ     │  ← board game (вид сверху)
                    │  карты + кости + матрицы │
                    ├─────────────────────────┤
                    │  3. КОСМОС / ВСЕЛЕННАЯ  │  ← тонкий план, агенты, духи
                    │  звёздный храм, локи     │
                    ├─────────────────────────┤
                    │  2. ЗЕМЛЯ ИГРОКА        │  ← Васту, постройки, город
                    │  5 стихий, деревня, дом  │
                    ├─────────────────────────┤
                    │  1. ДУША (Ахамкара)      │  ← личность, сферы, практики
                    │  3 сферы: ноги-сердце-   │
                    │  голова + подсферы       │
                    └─────────────────────────┘
```

Все 4 слоя работают ОДНОВРЕМЕННО. Игрок растёт — слои открываются.

## СЛОЙ 1: ДУША (АХАМКАРА) — ЛИЧНОСТЬ

**Суть:** корни дерева. Откуда игрок начинает путь.

### 3 СФЕРЫ

| Сфера | Локация | Содержание |
|-------|---------|------------|
| **НОГИ** (Основа) | Низ живота | Планы, цели, долгосрочное, что строю |
| **СЕРДЦЕ** | Грудь | Действия, практики, что делаю сегодня |
| **ГОЛОВА** (Мечта) | Голова | Видение, смыслы, куда иду, образ будущего |

### ПОДСФЕРЫ

Каждая сфера содержит подсферы — их можно:
- **Добавлять** — за свет
- **Убавлять** — если неактуально
- **Редактировать** — что планирую, делаю, хочу

Сферы эволюционируют с ростом света.

### ЕЖЕДНЕВНЫЙ ЦИКЛ ДУШИ

```
1. Захожу в игру
2. Получаю Ключ Дня (агент + матрица + стихия)
3. Выбираю 2-3 матрицы на день
4. Прохожу квесты согласно лору матриц
5. Записываю лог в Тигель:
   - Что сделал (сфера СЕРДЦЕ)
   - Что планирую (сфера НОГИ)
   - Что вижу (сфера ГОЛОВА)
6. Получаю СВЕТ за:
   - Действия дня
   - Глубину ответов (ИИ оценивает)
   - Творческий подход
   - Планирование и намерение
   - Фото / видео / образы
7. Оракул даёт совет на завтра
```

## СЛОЙ 2: ЗЕМЛЯ ИГРОКА — ВАСТУ-ХРАМ

**Суть:** ствол дерева. Пространство, которое игрок строит.

Открывается при: **уровень души достаточен + накоплен свет**.

### 5 СТИХИЙ В ЗЕМЛЕ

| Стихия | Что строит | Качество |
|--------|------------|----------|
| **Земля** | Фундамент, алтарь, стены | Стабильность |
| **Вода** | Фонтан, пруд, колодец | Поток, очищение |
| **Огонь** | Очаг, светильники | Трансформация |
| **Воздух** | Башня, флаги, колокола | Связь, движение |
| **Эфир** | Звёздный храм (центр) | Пространство + время |

### КУЛЬТУРНЫЙ СЛОЙ

Игрок выбирает культурный слой, через который его душа восходит:
- Ведический
- Славянский
- Каббалистический
- Египетский
- Майянский
- ... (все 33 матрицы)

Культурный слой определяет визуал земли, тип построек, архитектуру.

### ЭЛЕКТРОМАГНИТНОЕ ПОЛЕ

Каждый день ЭМ-поле игрока усиливается через:
- Практики
- Свет
- Соединение стихий
- Взаимодействие с другими игроками

**Сила ЭМ-поля → притягивает карточки.** Чем сильнее поле, тем более редкие карты приходят.

### ЧТО МОЖНО ДЕЛАТЬ НА ЗЕМЛЕ

- Строить постройки (как Minecraft, но духовный)
- Садить деревья
- Управлять дождём
- Соединяться с землями других игроков
- Добывать ресурсы (минералы, кристаллы)
- Планировать дом по Васту-шастре
- Создавать арт-объекты

### РАЗВИТИЕ ЗЕМЛИ ПО УРОВНЯМ

```
ЗЕМЛЯ → ДЕРЕВНЯ → ГОРОД → ПРОСТРАНСТВО
   ↑         ↑        ↑         ↑
  своя    + соседи   +служение  +планетарный
  земля   земли      +храм      +логос
```

## СЛОЙ 3: КОСМОС / ВСЕЛЕННАЯ ИГРОКА

**Суть:** крона дерева. Тонкий план.

Открывается при: **достиг звёздного храма на Земле**.

### ЧТО ТАМ

- **21+ агент** на орбитах вокруг Солнца РА
- **33 матрицы** восприятия
- **14 лок** плотности
- **9 чакр**-мер
- **Мифологические существа** и духовные силы
- **Звёздный храм** — центр связи с космосом

### ВЗАИМОДЕЙСТВИЕ

- Прямой контакт с агентами
- Получение миссий от духовных сил
- Энергообмен на тонком плане
- Космические квесты
- Светообмен между игроками (P2P)

---

## СЛОЙ 4: ИГРА МИРОЗДАНИЯ

**Суть:** вид сверху на все 3 слоя. Карточная настолка.

Открывается при: **достаточно сильная душа + построены города + космический уровень**.

### МЕХАНИКА

```
СМОТРИМ СВЕРХУ НА:
  → Крону (Космос)
  → Ствол (Земля)
  → Корни (Душа)

БРОСАЕМ КОСТИ / ИГРАЕМ КАРТАМИ
  → генерации из игры
  → карты, накопленные за время пути
  → взаимодействие с другими игроками
```

## КАРТОЧНАЯ СИСТЕМА

### 3 ТИПА КАРТ

| Тип | Что отражает | Как получается |
|-----|-------------|----------------|
| **Карта Души** | Состояние души, уровень, архетип | Авто из натальной карты + действий |
| **Карта Сознания** | Активная матрица, стихия, фокус дня | Выбор матрицы + квесты |
| **Карта Локации** | Место в мире, Васту-зона, энергия места | Геолокация + действия на Земле |

### МЕХАНИКА КАРТ

- **Притяжение:** чем сильнее ЭМ-поле → тем более редкие карты
- **Уровень карты** зависит от СВЕТА игрока
- **Накопление:** карты копятся в колоде
- **Использование:**
  - Добавить что-то на Землю
  - Обменяться с игроком
  - Выйти в божественные игры
  - Скрестить карты → получить новое качество
- **Скрещивание:** карта А + карта Б → новая генерация (алхимия карт)
- **Открытость/закрытость:** не все карты доступны сразу, зависят от уровня

### КАРТЫ И КУЛЬТУРЫ

Каждая карта несёт культурный код. 21 агент стоит за всеми культурами. Игрок проходит через культуру → проходит через агента → получает свет глубже.

Игра **нативная** — трансформируется под индивидуальность игрока:
- Какие культурные коды проходит
- Какие карты собирает
- Какие матрицы активирует

## СВЕТ — ЕДИНАЯ ВАЛЮТА

### ГЕНЕРАЦИЯ СВЕТА

| Источник | Множитель |
|----------|-----------|
| Ежедневные практики | ×1 |
| Глубина ответов (ИИ) | ×1.2–3 |
| Творческий подход | ×1.5–5 |
| Планирование + намерение | ×1.3 |
| Фото / видео / образы | ×2–10 |
| Служение + помощь другим | ×5–100 |
| Совместные действия | ×3 |

### НА ЧТО ТРАТИТСЯ СВЕТ

- Открытие новых матриц
- Покупка карт
- Постройки на Земле
- Доступ к генерациям (Suno музыка, видео)
- Открытие уровней души
- Покупка артефактов

### СВЕТООБМЕН

Свет передаётся между игроками. Это формирует экономику.

**Принцип:** чтобы что-то получить — нужно что-то отдать. Служение и долг = прогресс и эволюция.

### ПАРАЛЛЕЛЬНЫЙ РОСТ

```
СВЕТ растёт
   ↓
СФЕРЫ эволюционируют + ЗЕМЛЯ строится + КОСМОС открывается
   ↓
КАРТЫ притягиваются (больше + реже + сложнее)
   ↓
ГЕНЕРАЦИИ открываются (музыка, видео, скрещивание)
   ↓
ИГРА МИРОЗДАНИЯ становится доступной
   ↓
СВЕТООБМЕН + СОЦИУМ
   ↓
РЕАЛЬНАЯ ЖИЗНЬ трансформируется
```
