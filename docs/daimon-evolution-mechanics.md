## 2. Главная формула развития Daimon

```txt
Daimon XP поднимает уровень.
Уровень открывает чакры.
Чакры подводят к грантхи.
Грантхи открывают новую стадию.
Стадия меняет форму и способности.
Способности открывают синтез, карты, домены и миры.
```

Daimon не должен расти только от кликов. Он должен расти от качества опыта игрока:

```txt
Тигель → честное проживание → очищение → свет → опыт Daimon → новая форма
```

## 3. Большая шкала эволюции

### Stage 1 — Пашу / Малыш / Пробуждение

```txt
Уровни: 1–20
Tier: Common
Чакры: 1–2
ДНК: 2 нити
Грантхи: нет
Роль: искра, детёныш, первый спутник
```

Механика:

- чувствует настроение игрока
- получает XP за Тигель
- даёт малый бонус к свету
- реагирует на совпадение стихии дня

Для Златогроза:

```txt
маленький белый кристальный котёнок-барс,
едва видимые золотые рожки,
маленькие синие искры Ардры.
```

---

### Stage 2 — Вира / Юный спутник / Герой

```txt
Уровни: 21–40
Tier: Uncommon
Чакры: 3–4
ДНК: 4 нити
Грантхи: Брахма-грантхи
Роль: спутник-воин, эмоциональный резонатор
```

Открывает:

```txt
seeEmotions
compositeElement
```

Механика:

- видит эмоциональные поля
- усиливает честный Тигель
- открывает двойные синтезы стихий
- начинает помогать превращать боль в свет

Для Златогроза:

```txt
юный снежный барс,
видимые золотые рожки,
кристаллический гребень,
электрические следы на лапах.
```

---

### Stage 3 — Садхака / Зрелый Daimon / Практикующий

```txt
Уровни: 41–60
Tier: Rare
Чакры: 5–6
ДНК: 7 нитей
Грантхи: Вишну-грантхи
Роль: защитник, голос, мантра, распознавание иллюзии
```

Открывает:

```txt
mantra
seeThroughMaya
```

Механика:

- персональная мантра Daimon
- подсказки через карты, сны, символы
- видит искажения дня
- усиливает карты по агентам, матрицам и стихиям
- даёт бонус за работу с shadow/distortions

Для Златогроза:

```txt
полный кристальный белый тигр / барс,
огненное сердце Агни,
грозовая грива,
голос-мантра.
```

---

### Stage 4 — Дживанмукта / Архитектор домена

```txt
Уровни: 61–80
Tier: Epic
Чакры: 7–8
ДНК: 10 нитей
Грантхи: Рудра-грантхи
Роль: строитель домена, читатель Акаши, тройной синтез
```

Открывает:

```txt

domainLink
```

Механика:

- можно соединять 3 стихии
- можно создавать domain_seed
- Daimon привязывается к домену игрока
- читает Акашический след события
- может стабилизировать созданные сущности

Для Златогроза:

```txt
огромный бело-кристальный тигр-хранитель,
молнии Ардры вокруг тела,
чакровые кольца за спиной,
золотой огонь Криттики в рогах,
под лапами возникает кристаллический храм.
```

Уровни: 81–100
Tier: Mythic / Legendary
Чакра: 9
ДНК: 12 нитей

## 4. Источники XP

### 1. Тигель

Главный источник роста Daimon.

```txt
обычная запись: +20 XP
глубокая запись: +50 XP
совпадение стихии: x1.2
работа с искажением: x1.5
```

Для Ардра-Daimon типа Златогроза нужно особенно награждать:

```txt
честность
кризис
слёзы
признание боли
перевод хаоса в смысл
```

---

### 2. Карты

Daimon получает XP, если карта совпадает с ним по:

```txt
стихии
агенту
матрице
архетипу
накшатре / теме
```

Для Златогроза особенно подходят:

```txt
Шива / Рудра
Свет Ра / Агни
Сарасвати
Брахма / Вишвакарман
Лакшми
Ведическая матрица
Герметико-алхимическая
Шамбала
Gene Keys
Каббала
```

---

### 3. Синтез

XP за участие в синтезе:

```txt
resonance_form: +100 XP
elemental_minion: +300 XP
artifact: +500 XP
domain_seed: +1000 XP
shared_domain: +3000 XP
```

---

### 4. Грантхи-квесты

Переход между большими стадиями требует не только XP, но и инициации.

```txt
Level 20 → Брахма-грантхи
Level 40 → Вишну-грантхи
Level 60 → Рудра-грантхи
Level 80 → выход за Брахманду
```

---

## 5. Грантхи как ворота

### Брахма-грантхи

```txt
Переход: Stage 1 → Stage 2
Тема: тело, страх, выживание, инерция
Квест: сделать действие в материи
```

Для Златогроза:

```txt
признать боль,
заземлить бурю,
не убегать из тела.
```

### Вишну-грантхи

```txt
Переход: Stage 2 → Stage 3
Тема: сердце, привязанности, отношения, голос
Квест: сказать правду без разрушения
```

Для Златогроза:

```txt
превратить грозу в мантру.
```

### Рудра-грантхи

```txt
Переход: Stage 3 → Stage 4
Тема: эго, контроль, образ себя, страх растворения
Квест: отпустить ложную форму
```

Для Златогроза:

```txt
разрушить ложный храм,
построить настоящий.
```

### Выход за Брахманду

```txt
Переход: Stage 4 → Stage 5
Тема: со-творчество
Квест: создать домен / помочь другим / завершить цикл Шивы
```

## 8. Ключевая идея

Daimon — это не питомец. Это путь игрока в форме существа.

```txt
Малыш отражает начало пути.
Юный спутник отражает первые победы.
Зрелый Daimon отражает голос и силу.
Архитектор отражает способность строить мир.
Космический Daimon отражает со-творца.
```

Для Златогроза конкретно:

```txt
детёныш бури
→ грозовой спутник
→ кристальный защитник
→ архитектор храма
→ космический Рудра-тигр
```
