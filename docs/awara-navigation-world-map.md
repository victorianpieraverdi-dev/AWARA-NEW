# AWARA — Карта экранов, навигации и миров

Дата фиксации: 2026-06-07
Источник: голосовое/текстовое описание + схема на фото.
Статус: концепт сохранён. Требует дальнейшей декомпозиции в UX, data model и задачи для реализации.

---

## 0. Главная идея

AWARA должна быть не набором страниц, а живой навигационной космологией.

Игрок перемещается по экранам как по пространству:

```txt
влево / вправо / вверх / вниз
свайпы на смартфоне
кнопки-направления на экране
плавные переходы через окна/порталы, не просто слайды
```

Структура нумерации:

```txt
0
1
1.1
1.2
1.3
1.4
2
3
4
5
6
7
...
```

Нумерация должна стать картой всей игры.

---

## 1. Навигация

### Требования

- На смартфоне и сайте должны быть направления:
  - влево
  - вправо
  - вверх
  - вниз
- Управление:
  - кнопками по краям экрана
  - свайпами
  - кликами по объектам
- Переходы должны быть плавными:
  - через окно
  - через портал
  - через приближение/погружение
  - через раскрытие сферы

### Центральная идея

```txt
Игрок не листает страницы.
Игрок путешествует по внутренней космологии AWARA.
```

## Образ

Экран входа — тёмный мистический космос.

В центре:

```txt

```

Фон:

```txt
тёмный космос

слабое мистическое свечение
ощущение тайны
```

## Поведение

При наведении/приближении:

```txt
внутри появляется Искра Света
точка света пульсирует
она зовёт нажать дальше
```

При клике:

```txt
0 → 1
```

После первого прохождения экран 0 может пропускаться.
Игрок может сразу попадать в:

```txt
1 — личная Вселенная
или
2 — центральное лобби
```

# 2 — Центральное лобби / Яйцо / Главное меню

## Суть

Главный центр игры.

Образ:

```txt
яйцо

7 или 9 энергоцентров

```

## Четыре главных направления

Из центра 2:

```txt
вправо → 3 — Хроника / информация / матрицы
вниз → 4 — Daimon / персональное существо
влево → 5 — Душа / игра / энергия дня
вверх → 6 — Световая настольная игра / карты
```

## Дополнительные кнопки вокруг яйца

```txt
слева сверху — уровень игрока / dashboard
справа сверху — меню
справа снизу — чат / связь с игроками / гильдия
слева снизу — мини-игры
```

## Поведение меню

Изначально яйцо может быть пустым.
При нажатии на меню внутри яйца появляются 4 сектора:

```txt
3 / 4 / 5 / 6
```

## Рост центрального окна

Центральное окно должно трансформироваться по мере роста Света игрока.

Чем больше света:

```txt
больше объёма
больше прозрачности
больше подвижности
можно переносить окна
можно открывать дополнительные сферы
открываются X/Y/Z-пространства
плоская карта становится объёмной
```

# Голос Совести / Голос Истины / Разум Игры

## Главная фундаментальная система

Нужно создать голос:

```txt
голос совести
голос правды
голос истины
голос разума
голос игры
```

Он распределяет энергию дня.

## Главный сценарий

Игрок записывает день одним аудиофайлом или текстом.

AI/Голос Игры:

```txt
слушает день
переспрашивает
уточняет
распределяет энергию по сферам
показывает, где личность/эго
показывает, какая энергия проявилась
```

## Куда голос распределяет энергию

```txt
Daimon
настольная игра
душа
локация
тонкий план
карты
агенты
матрицы
храм
космос игрока
```

## Уточняющие вопросы

Голос должен спрашивать:

```txt
Хотите уточнить по тонкому плану?
Хотите уточнить по локации?
Было ли содействие с другими игроками?
Какие сферы были задействованы?
Какая энергия проявилась?
```

## Важное

Разум игры должен усложняться вместе с сознанием игрока.

На ранних уровнях:

```txt
простые подсказки
простые действия
меньше интерпретаций
```

На высоких уровнях:

```txt
глубокий разбор
матрицы
агенты
карты
домены
космические связи
```

# 3 — Хроника / Информация / Матрицы

## Суть

Если 5 — это энергия души, то 3 — это информация, структура знания, хроника.

Здесь должно отображаться:

```txt
сегодняшний день
звёзды
астрологические влияния
выбранные матрицы
натальная карта игрока
Daimon
накопления игрока
полученные ключи
состояние прохождения матриц
```

## 3.1 — Матрицы и ключи

У нас есть:

```txt
33 матрицы
21 агент
693 связи агент × матрица
```

Игрок проходит матрицы и получает ключи.

Матрицы влияют на:

```txt
культуру сознания
духовную индивидуальность
космос игрока
храмы
карты
Daimon
```

## 3.2 — Сундуки / кейсы / карты силы

Ключи открывают:

```txt
сундуки
кейсы
карты силы
божественные карты
```

Эти карты потом используются в 6 — настольной игре.

## 3.3 — Индивидуальное духовное сознание

Из пройденных матриц, карт, ключей, артефактов формируется духовная индивидуальность игрока.

```txt
мультикультурность
мульти-вселенная
индивидуальный духовный профиль
```

## Артефакты времени

Игрок может получать:

```txt
кристаллы времени
артефакты времени
древние артефакты силы
```

Чем больше светимость и тонкость игрока, тем выше шанс получить такие артефакты.

# 6 — Световая настольная игра / Карты / Центр Ра

## Суть

Это главный бриллиант игры.

Настольная игра должна собрать в себя все смыслы AWARA.

Образ:

```txt
многокольцевая мандала
путь от периферии к центру
в центре — Ра / Источник
```

## Путь

От грубого к световому:

```txt
чёрный / серый / бурый / красный
→ оранжевый
→ жёлтый
→ зелёный
→ голубой
→ синий
→ фиолетовый
→ серебряный
→ золотой
→ центр Ра
```

## Слои игры

В настолку должны входить:

```txt
7 энергоцентров
9 измерений
14 лок
агенты
матрицы
позитивные/негативные проявления
время
знаки Зодиака
накшатры
Daimon
карты силы
световые карты
```

## Смысл

Игрок идёт от периферии к центру.

Пример уровней:

```txt
1 — собственный ум
2 — психоэмоциональные состояния
3 — способность строить структуру
4 — духовная индивидуальность и сотрудничество
5 — сердце / человеческое взаимодействие
6 — служение
7 — космическое созерцание
8 — выход за земной план
9 — космический игрок
10 — возвращение в Источник
```

## 6.1 — Додекаэдр / Планетарный логос

Настольные игры разворачиваются в точках планетарного логоса.

Идея:

```txt
62 основные точки планетарного логоса
чакры планетарного логоса
каждая игра разворачивает портал
каждая точка продышивается

```

## 6.2 — Игроки / карты других игроков


# Детские / семейные мини-игры

## Кукольный театр / игрушки / глина

Идея:

```txt
каждая игрушка имеет историю
каждая история учит космическому принципу
архетип животного проходит тень → дар → сиддхи
история разыгрывается через кукол/глину/предметы
```

## 12 записок / поэтические квесты

Игроки/дети идут по карте через стихи и подсказки.

Пример формата:

```txt
Ты пришёл отсюда, мастер.
Дальше иди туда, где ветер ведёт и не оставляет следов.
```

## Значки / медали / звания

За прохождение квестов:

```txt
земля 1–6 уровня
вода 1–6 уровня
огонь 1–6 уровня
воздух 1–6 уровня
звёздный строитель
мастер стихии
```

Скрещивание стихий:

```txt
земля 5 + воздух 6 → новое звание / артефакт / способность
```


```txt
позитивные варианты
выходы из состояния
практики восстановления
подсветку тёмных мест
маленькие действия
```

# MVP-порядок реализации

## P0 — Навигационная карта

Создать data/awara_navigation_map.json с узлами:

```txt
0, 1, 1.1, 1.2, 1.22, 1.23, 1.3, 1.4, 2, 3, 4, 5, 6, 7
```

## P1 — Центральное лобби 2

Сделать экран яйца:

```txt
центр
4 направления
кнопки вокруг
свайпы
плавные переходы
```

## P2 — Экран 0

Сделать вход:

```txt
тёмный космос
Искра Света
пульсация
переход в 1 или 2
```

## P3 — Экран 1

Сделать персональную Вселенную:

```txt
рандомный тороид
светимость от totalLight
листание других игроков позже
```

## P4 — Голос Игры / распределение дня

Сделать простой intake:

```txt
запись дня
AI-разбор
распределение в душу / Daimon / карты / локацию / тонкий план
```

## P5 — Экран 5 Душа

Сделать сферы души и запись дня.

## P6 — Экран 4 Daimon

Связать с уже созданной системой прогресса Daimon.

## P7 — Экран 3 Матрицы

Показать день, матрицы, ключи, хронику.

## P8 — Экран 6 Настолка

Начать с карты/коллекции, не делать сразу полную настольную игру.

## P9 — Экран 7 Полевая игра

Сохранить как будущий live/festival layer.

