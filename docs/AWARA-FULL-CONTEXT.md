## 1. Что такое AWARA

AWARA — Сакральная Операционная Система. Это одновременно эзотерическая игра, духовный инструмент и ОС сознания. Цель — провести игрока через путь от Кали-Юги к Сатья-Юге, трансформируя повседневную жизнь в осознанную практику через призму 33 мировых духовных традиций.

## 2. Философия

- **«Игра — это состояние. Состояние игры.»** Цель не контент, а ввод игрока в состояния сознания, их синтез и углубление.
- **«Выполнение = свет.»** Любое выполненное действие/практика конвертируется в частицы света — универсальную внутриигровую валюту.
- **«Всё включено, всё разрешено»** — Супер-Игра. Никаких жёстких рельсов: всё многообразие практик, культур, генераций доступно по мере накопления света.
- **Формула:** Внимание → Опыт → Осознание → Свет → Карта → Структура → Мир
- **Тело** — первая «живая таблица». Сначала физический отклик, затем эмоциональный резонанс, потом ментальная конструкция.

## 3. Архитектура: 4 слоя игрока

```
┌─────────────────────────────┐
│  4. ИГРА МИРОЗДАНИЯ         │  ← настольная игра (вид сверху)
│     карты + кости + матрицы │
├─────────────────────────────┤
│  3. КОСМОС / ВСЕЛЕННАЯ      │  ← тонкий план, агенты, духи
│     звёздный храм, локи     │
├─────────────────────────────┤
│  2. ЗЕМЛЯ ИГРОКА            │  ← Васту, постройки, город
│     5 стихий, деревня, дом  │
├─────────────────────────────┤
│  1. ДУША (Ахамкара)         │  ← личность, сферы, практики
│     3 сферы: ноги-сердце-   │
│     голова + подсферы       │
└─────────────────────────────┘
```

Все 4 слоя работают одновременно. Слои открываются по мере роста игрока.

## 9. Даймон — внутренний спутник

Даймон — персональная сущность игрока, его внутренний ребёнок / искра / будущий Бог. Форма Даймона определяется по накшатре (лунной стоянке) из натальной карты.

### 4 стадии эволюции Даймона

| Стадия | Имя | Нити ДНК | Чакры | Условие |
|--------|-----|----------|-------|---------|
| 1 | Пашу (Связанная Душа) | 2 | 1–2 | Старт |
| 2 | Вира (Герой) | 4 | 3–4 | Пробой Брахма-грантхи |
| 3 | Садхака (Практикующий) | 7 | 5–6 | Пробой Вишну-грантхи |
| 4 | Дживанмукта (Освобождённый) | 10 | 7–8 | Пробой Рудра-грантхи |

### Формы Даймона (по накшатрам)

Олень (Ашвини), Слон (Бхарани), Феникс (Криттика), Лебедь (Рохини), Лань (Мригашира), Волк (Ардра) и т.д. — **27 форм** по накшатрам.

Даймон даёт до 10 советов, имеет характер, арку речи, живёт в своих Царствах (минералы, растения, TBD).

_Примечание: в части кода/доков CLEAN встречается 7 стадий Даймона — при расхождении сверять с `daimon-stages.json` и решением Павла._

## 10. Экономика света

- Частицы света — универсальная внутриигровая валюта
- Эфир — пробуждается действиями
- Свет получают за: действия дня, глубину ответов, творческий подход, планирование, фото/видео/образы
- За частицы света открываются: кейсы (фрагменты карт + артефакты + бонусы), новые уровни, контент

### Готовые модули экономики

- `awara-particles.js` — частицы + эфир
- `awara-sensitivity.js` — чувствительность + точка сборки
- `awara-light-structures.js` — световые структуры
- `awara-passive-flow.js` — пассивный поток + чекин
- `awara-rarity.js` — раритет промптов
- `awara-scroll.js` — свиток
- `awara-longterm.js` — панорама 33 линзы
- `awara-energy-map.js` — энергокарта на завтра

## 11. Ежедневный цикл

1. Захожу в игру
2. Получаю **Ключ Дня** (агент + матрица + стихия)
3. Выбираю 2–3 матрицы на день
4. Прохожу квесты согласно лору матриц
5. Записываю лог в **Тигель**:
   - Что сделал (сфера СЕРДЦЕ)
   - Что планирую (сфера НОГИ)
   - Что вижу (сфера ГОЛОВА)
6. Получаю **СВЕТ** за действия
7. **Оракул** даёт совет на завтра
