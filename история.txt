Задача: визуальная трансформация карточек колоды (deck cards) по уровням L1→L6 в AWARA Тигель.

== ЧТО СДЕЛАНО (не трогай, работает) ==

Файл `C:\AWARA\awara-card-visuals.js` — единственный источник визуальных стилей карточек.
- Standalone скрипт, подключён в tigel-app.html: <script src="awara-card-visuals.js?v=5">
- Использует setProperty(..., 'important') чтобы гарантированно перебить CSS
- MutationObserver на #deck + click listener на #lvTestBtns + setTimeout fallbacks
- Палитра: window.lensPalette(key) → [c1, c2, c3] — уникальные цвета каждой из 33 традиций
- Уровень: window.__deckLvPreview (тест-панель) или AwaraAscension.level(key)
- Орб тинтинг: .cv-tint div с mix-blend-mode:color внутри .gl
- Размер панелей НЕ МЕНЯЕТСЯ (transform:none!important на всех уровнях)

В `awara-lens.js` (v=31) стилизация в decorateDeck отключена через if(false){...}.
Оставлены: paintOrb, data-alv, lens-cl-*, lv-badge, lens-badge.
Также есть CSS-правила [data-alv] в начале файла — они перебиваются !important из awara-card-visuals.js.

== ЧТО НАДО СДЕЛАТЬ ==

1. ЦВЕТ ПЛАШЕК — сейчас фоны карточек почти прозрачные (opacity .06-.16). Надо сделать заметный цветной фон используя c1/c2/c3 палитры традиции. Рост от L1 к L6 = рост насыщенности и глубины цвета, НЕ размера.

2. УСЛОЖНЕНИЕ АРХИТЕКТУРЫ ПАНЕЛИ от L1 к L6:
      - L1: минимальная, плоская, ч/б, едва видно
      - L2: появляется лёгкий цвет и тонкий бордер
      - L3: градиент, заметный цвет, лёгкое свечение
      - L4: двухцветный градиент (c1+c2), чёткий бордер, внутреннее свечение
      - L5: добавляются декоративные элементы (уголки, паттерн, тонкая рамка), богатый цвет
      - L6: полная роскошь — многослойный градиент, свечение, декор, ощущение завершённости
   
   Идея: от плоскости к объёму, от пустоты к наполненности. Качество, не количество.

== КРИТИЧЕСКИЕ ТЕХНИЧЕСКИЕ ОГРАНИЧЕНИЯ ==

1. Все стили ставить через setProperty('prop', value, 'important') — иначе CSS из awara-lens.js перебьёт
2. transform: none !important — ВСЕГДА, размер панелей не должен меняться
3. decorateDeck в awara-lens.js — НЕ ТРОГАТЬ (уже отключен if(false)), он ставит стили БЕЗ !important и перезаписывает
4. paintOrb заменяет innerHTML .gl — .cv-tint div пропадает. MutationObserver пересоздаёт его
5. MCP edit на tigel-app.html (~100KB) таймаутит — редактируй только awara-card-visuals.js
6. Не забывай бампить ?v=N в tigel-app.html при каждом изменении
7. Юзер должен делать Ctrl+Shift+R после каждого изменения

Ясно. Изучил engine_config.json v2.3 целиком — все 25 секций, включая v2.3b обновления от Русского Мастера. Полностью понял идеальный цикл. Вот он как я его вижу:
1:20
Слой 1 — ПОЗНАИ СЕБЯ (3 уровня глубины)
A) Натал → Даймон
Дата/время/город → сидерическая карта (Лагна, 9 планет, накшатры, пады) → Даймон из накшатры Луны (имя, стихия, архетип). Игра знает твой космический отпечаток.
B) Профиль → Линзы + Оси
9 сфер, 46 путей → стартовые линзы (maps_to_lens), доминантные оси, стартовая лока. engine_config добавляет: каждый путь мапится на локу, чакру, ось — профиль сразу влияет на всю экономику. Игра знает твой направления.
C) Свиток (Прошлое / Настоящее / Будущее)
Natal + Profile + Scroll = тройная глубина. Свиток добавляет контекст: откуда идёшь, где сейчас, куда стремишься. Игра знает тебя как человека.
1:20
Слой 2 — ЗНАКОМСТВО + НАМЕРЕНИЯ
Даймон приветствует → показывает плашки, подсказки, инструкции — игрок осваивается.
Намерения (через intention_classification + life_activities):
Игрок пишет свободным текстом:
«Бегать по утрам» → ИИ находит running в каталоге 75 деятельностей → type=discipline, линзы: [norse, slavic], оси: [discipline, will]
«Йога по вечерам» → yoga_hatha → линзы: [vedic, tantric_kashmiri], оси: [discipline, will]
«Мантра Ом перед сном» → mantra → линзы: [vedic, buddhist], оси: [devotion, clarity]
Главное: даже если этих линз у тебя нет, они начинают накапливать пассивный свет (+0.1/день practice, +0.3/день discipline). Стрик усиливает: 8-21 день ×1.3, 22-60 ×1.6, 61+ ×2.0. Когда пассивный свет достигнет 5.0 — спящая линза пробуждается: «Твоя практика мантры пробудила линзу Ведическую».
1:21
Слой 3 — УТРО: ЭНЕРГИЯ ДНЯ + КВЕСТЫ
:sunny: Энергия дня (daily_energy) — перед действиями:
7 календарных систем (Тропическая, Китайская, Майя, Нумерология, Human Design, Славянская, Шамбала) → доминантная стихия дня + агент дня (Сурья/Чандра/Будха/Гуру/Шукра/Шани/Мангала). Плюс натальные грани — положения планет игрока добавляют персональную окраску.
Результат: энергопортрет, гуны, оси, совет по кольцу. Загружается в сознание перед действиями.
Квесты — два источника:
От матриц (matrix_quests) — 2475 квестов с 7-осевым reward, уровни L1-L6, 7 типов (do, meditate, observe, reflect, create, study, ritual)
От локаций (location_quests) — 3033 квеста с 7-осевым reward
День: выполняешь квесты (cap по кольцу: ring -3 = 5 шт, ring 9 = 12 шт), практики, намерения.
daily_checkin при первом входе — проверка самодисциплин: «Бегал утром?» :heavy_check_mark: → streak +1, пассивный свет на линзы + оси.
1:22
Слой 4 — ВЕЧЕР: ОТЧЁТ + ИИ-ОЦЕНКА + МУЛЬТИПЛИКАТОРЫ
Отчёт: текст + фото/аудио/видео → идёт в runtime_flow (10 шагов).
ИИ-оценка (ai_evaluation) — Даймон анализирует текст и выдаёт:
element — доминантная стихия (Земля/Вода/Огонь/Воздух/Эфир)
guna — саттва/раджас/тамас
quality_score — искренность, глубина (0-1)
fullness_score — сколько уровней затронуто (тело+ум+чувства+воля+дух)
loka — 1-14 (от Сатья до Патала)
shadow_detected — проработанная тень (+10% света!)
creativity_level — none / lyrical / poetry / art_reference / original_creation
Мультипликаторы (8 множителей!):
base_reward × lens_depth(L1=1.0…L6=2.0) × daily_energy(×1.15-2.1) × agent(×1.2) × loka_density × guna(саттва=1.5) × quality × mastery(ring+quality_avg, до ×2.0) × creativity(до ×1.7) × media(до ×1.6)
Потом ИИ помогает распределить по линзам и шкале опыта (element_to_axes: Земля→дисциплина+воля, Вода→сострадание+преданность и т.д.). В dev_mode игрок может помочь с этим.

Слой 7 — НОЧЬ: КОНВЕРТАЦИЯ + ЗАВТРА
Частицы → Свет (ночной процесс):
Частицы дня × ether_conversion_rate (70-90% по ring) → свет
cascade_down: свет из высших окон Меры стекает вниз (10% каждому нижнему)
Пассивный поток от практик/дисциплин → линзы
Пересчёт Ring (пороги: 0→5→15→30→50→100→200→400→700→1200→2000→3500→6000)
Эфир → sleeping (если нет Якоря Вечности — световая структура, 1000 света на постройку, сохраняет 20% эфира)
Карта на завтра — перед сном:
Энергия завтрашнего дня + 3 квеста preview + совет по кольцу → загружается в сознание. Завтра — всё сначала.
Итого: полный цикл
Натал → Профиль → Свиток → Намерения → Энергия дня → Квесты → Отчёт+фото → ИИ-оценка → 8 мультипликаторов → Частицы → Мера → Арт+Трек (раритет!) → Ночь: cascade → Карта завтра → Сон → Повтор

Перед очисткой кэша
