# Подсферы и Тигель — прогресс 29-30 мая 2026

## Контекст
Пользователь хотел:
1. Подсферы должны быть **отдельными перемещаемыми объектами**, связанными с основной сферой
2. Клик на подсферу → открывает **её собственную панель** (не основной сферы)
3. Удобное создание: кнопка "+" → пустая подсфера → клик → наполнить → drag
4. Тигель: упростить кнопки (но потом откатили — вернули как было)

## Что сделано

### 1. Подсферы — независимые объекты (PR #188)
**Ветка:** `fix/subsphere-drag-and-buttons`  
**PR:** https://github.com/victorianpieraverdi-dev/awara-game/pull/188

#### Коммиты:
- `51999e4` — восстановлены кнопки "+", исправлен drag, улучшен визуал
- `6ad4db4` — отдельная панель для каждой подсферы + миграция старых

#### Изменения:
1. **Миграция старых подсфер:**
   - При загрузке проверяются все подсферы
   - Если нет `x, y` → добавляются орбитальные позиции
   - Теперь старые подсферы можно перетаскивать

2. **Drag подсфер:**
   - Добавлена переменная `pressSubsphere` для отслеживания нажатия
   - `pointermove` проверяет движение >8px и начинает drag
   - `pointerup` сохраняет позицию
   - `findSubsphere` использует сохранённые x,y вместо орбитальных

3. **Кнопки "+" восстановлены:**
   - Рядом с основной сферой
   - Клик → `createEmptySubsphere()` создаёт пустую подсферу
   - Счётчик: заполнено/доступно (например "1/2")

4. **Отдельная панель для подсферы:**
   - HTML: `<div id="subsphere-panel">` после основной панели
   - Функции: `openSubspherePanel()`, `closeSubspherePanel()`, `saveSubsphereEdit()`, `deleteSubsphere()`
   - Показывает: текст, дата, свет, родительская сфера (↑ Душа)
   - Кнопки: 💾 СОХРАНИТЬ / 🗑 УДАЛИТЬ

5. **Визуал:**
   - Заполненная: 16px, яркая, свечение
   - Пустая созданная: 14px, призрачная, иконка ✎, пульсация
   - Линия связи: всегда от подсферы к основной

#### Файлы:
- `spheres-v2.html` — все изменения
